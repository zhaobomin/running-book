## 技术要点速览

### 跑姿三原则

1. **放松是效率的基础** - 只有需要用力的肌肉在用力
2. **稳定性比力量更重要** - 核心稳定，力量有效传递
3. **小而快比大而慢更好** - 高步频，小步幅更高效

### 快速自检

```
👀 目光平视前方 10-20 米
👂 肩膀放松下沉
💪 手臂弯曲 90°
✋ 手掌放松微握
🫁 核心收紧
🦵 髋部稳定
🦶 脚掌中前部着地，落地点在重心下方
```

---

*改善跑姿是一个渐进的过程，每次跑步关注一两个要点，逐渐养成正确的习惯。*
