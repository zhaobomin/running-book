# 心率训练

> 用数据指导训练

## 一、心率基础

### 1. 为什么用心率训练？

**好处**：
- 客观衡量强度
- 避免训练过度或不足
- 追踪体能进步
- 个性化训练区间

### 2. 心率基本概念

| 概念 | 说明 |
|------|------|
| 静息心率 | 早晨醒来最低心率 |
| 最大心率 | 运动时能达到的最高心率 |
| 储备心率 | 最大心率 - 静息心率 |
| 心率区间 | 不同强度的百分比范围 |

## 二、确定最大心率

### 1. 公式估算

**经典公式**：
```
最大心率 = 220 - 年龄
```

**更准确的公式**（Tanaka）：
```
最大心率 = 208 - (0.7 × 年龄)
```

**示例**：
| 年龄 | 经典公式 | Tanaka 公式 |
|------|----------|-------------|
| 30 岁 | 190 | 187 |
| 40 岁 | 180 | 180 |
| 50 岁 | 170 | 173 |

### 2. 实测最大心率

**方法一：操场测试**
```
热身：15 分钟轻松跑
主测试：
- 800m 稍快
- 400m 更快
- 最后 400m 全力冲刺
看心率表最高值

注意：需要身体状态良好，有风险
```

**方法二：上坡测试**
```
热身：15 分钟
主测试：
- 找一个 400-800m 的上坡
- 全力跑 2-3 次
- 每次间慢跑下坡
- 记录最高心率
```

### 3. 静息心率测量

**方法**：
- 早晨醒来，不起床
- 测量 1 分钟心跳
- 连续测量一周取平均
- 记录基准值

## 三、心率区间

### 1. 五区间模型

| 区间 | 名称 | 心率范围 | 训练目的 |
|------|------|----------|----------|
| Z1 | 恢复 | 50-60% 最大 | 恢复、热身 |
| Z2 | 有氧基础 | 60-70% 最大 | 建立耐力 |
| Z3 | 有氧耐力 | 70-80% 最大 | 节奏跑 |
| Z4 | 阈值 | 80-90% 最大 | 间歇跑 |
| Z5 | 极限 | 90-100% 最大 | 冲刺、比赛 |

### 2. 使用储备心率（更准确）

**Karvonen 公式**：
```
目标心率 = (储备心率 × 强度%) + 静息心率
```

**示例**（最大 180，静息 60）：
| 区间 | 强度 | 心率范围 |
|------|------|----------|
| Z1 | 50-60% | 120-132 |
| Z2 | 60-70% | 132-144 |
| Z3 | 70-80% | 144-156 |
| Z4 | 80-90% | 156-168 |
| Z5 | 90-100% | 168-180 |

### 3. 简化区间（体感对应）

| 区间 | 体感 | 能说话程度 |
|------|------|------------|
| Z1 | 非常轻松 | 完整对话 |
| Z2 | 轻松 | 完整对话 |
| Z3 | 中等 | 短句 |
| Z4 | 困难 | 只能说几个词 |
| Z5 | 非常困难 | 无法说话 |

## 四、心率训练方法

### 1. 每周区间分配

**80/20 原则**：
- 80% 训练在 Z1-Z2
- 20% 训练在 Z3-Z5

**典型周安排**：
| 训练 | 区间 | 占比 |
|------|------|------|
| 轻松跑 | Z1-Z2 | 60% |
| 长距离 | Z2 | 20% |
| 节奏跑 | Z3 | 10% |
| 间歇跑 | Z4-Z5 | 10% |

### 2. 不同训练的心率目标

**轻松跑**
```
目标区间：Z1-Z2（最大心率 65-75%）
心率：稳定，不波动
体感：可以轻松对话
```

**长距离跑**
```
目标区间：Z2（最大心率 70-75%）
心率：可能后程上升（心率漂移）
体感：轻松到中等
```

**节奏跑**
```
目标区间：Z3-Z4 下限（最大心率 80-85%）
心率：稳定
体感：舒适地困难
```

**间歇跑**
```
目标区间：Z4-Z5（最大心率 85-95%）
心率：高强度时达到，间歇期下降
体感：困难到非常困难
```

### 3. 心率漂移

**定义**：相同配速下，心率随时间上升

**原因**：
- 体温升高
- 脱水
- 疲劳

**应对**：
- 正常现象
- 可以稍降配速
- 保持心率在目标区间

## 五、心率监测设备

### 1. 心率带 vs 光学心率

| 类型 | 优点 | 缺点 |
|------|------|------|
| 心率带 | 准确、响应快 | 佩戴不适 |
| 光学心率 | 方便、无感 | 准确度稍差 |

### 2. 推荐设备

**心率带**：
- Garmin HRM-Pro
- Polar H10
- Wahoo TICKR

**运动手表**：
- Garmin Forerunner 系列
- Polar Vantage 系列
- Apple Watch
- Suunto

### 3. 使用技巧

- 戴紧心率带/手表
- 保持传感器清洁
- 冷天热身充分再开始
- 关注趋势，不只看单次

## 六、心率数据应用

### 1. 训练监控

**日常训练**：
- 是否在目标区间
- 心率是否稳定
- 有无异常波动

**疲劳信号**：
- 同配速心率更高
- 静息心率升高
- 心率恢复变慢

### 2. 体能评估

**进步指标**：
- 同配速心率更低
- 同心率配速更快
- 心率恢复更快

**测试方法**：
```
每个月做一次测试：
1. 热身 10 分钟
2. 以固定配速跑 10 分钟（如 5:30/公里）
3. 记录平均心率
4. 比较：心率下降 = 体能提升
```

### 3. 比赛策略

**半马/全马心率策略**：
```
前半程：控制在 Z3 下限（约 80% 最大心率）
后半程：允许上升至 Z3-Z4
```

## 七、常见问题

### Q1: 心率总是偏高/偏低怎么办？

**偏高**：
- 可能是测量误差
- 可能是疲劳、高温、脱水
- 调整体感训练

**偏低**：
- 可能是测量问题
- 可能是有氧基础好
- 不用担心

### Q2: 最大心率公式不准？

是的，公式只是估算。误差可能 ±10-15 次。
建议实测或结合体感判断。

### Q3: 应该看心率还是体感？

两者结合：
- 心率：客观参考
- 体感：主观感受
- 理想情况两者一致
