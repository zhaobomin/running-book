# 比赛策略

> 训练是准备，比赛是艺术

## 一、赛前准备

### 1. 赛前一月

**训练调整**：
- 跑量逐渐减少（减量期）
- 保持强度，减少量
- 长距离减少但不停

**减量计划**：
| 时间 | 跑量 |
|------|------|
| 赛前 3 周 | 80% |
| 赛前 2 周 | 60% |
| 赛前 1 周 | 40% |

### 2. 赛前一周

**训练**：
- 周一：休息
- 周二：轻松跑 6K
- 周三：轻松跑 5K + 2×400m 比赛配速
- 周四：休息
- 周五：轻松跑 4K
- 周六：比赛日

**饮食**：
- 碳水 loading（每公斤体重 7-10g 碳水）
- 减少纤维
- 多喝水
- 避免新食物

**睡眠**：
- 重点在赛前 2-3 晚
- 比赛前一晚紧张很正常
- 重要的是前几晚睡好

### 3. 比赛前一天

**准备清单**：
- [ ] 号码布别好
- [ ] 芯片绑好
- [ ] 比赛装备准备好
- [ ] 能量胶、盐丸准备
- [ ] 熟悉赛道和补给站
- [ ] 设定配速提醒
- [ ] 早餐食物准备

**活动**：
- 轻松活动，不要过度
- 领取参赛包
- 熟悉起点位置
- 早睡

### 4. 比赛日早晨

**时间安排**：
| 时间 | 活动 |
|------|------|
| 赛前 3 小时 | 起床、早餐 |
| 赛前 2 小时 | 轻微活动、喝水 |
| 赛前 1 小时 | 到达起点 |
| 赛前 30 分钟 | 热身、存包 |
| 赛前 10 分钟 | 进入起跑区 |

**早餐建议**：
- 碳水 100-150g
- 低纤维、低脂肪
- 熟悉的食物
- 如：面包+香蕉+运动饮料

## 二、比赛配速策略

### 1. 通用原则

**负分割**（后半程更快）：
- 最推荐的策略
- 前半程保守
- 后半程保持或加速

**原因**：
- 起跑太快要付出代价
- 后程崩溃比前程稍慢更痛苦
- 心理上更轻松

### 2. 5K 比赛策略

| 段落 | 配速建议 | 心态 |
|------|----------|------|
| 0-1K | 比目标慢 5-10 秒 | 压住冲动 |
| 1-3K | 目标配速 | 找到节奏 |
| 3-4K | 保持 | 坚持住 |
| 4-5K | 全力 | 冲刺 |

### 3. 10K 比赛策略

| 段落 | 配速建议 | 心态 |
|------|----------|------|
| 0-2K | 比目标慢 5 秒 | 稳定开始 |
| 2-5K | 目标配速 | 保持节奏 |
| 5-8K | 目标配速 | 最艰难阶段 |
| 8-10K | 全力 | 冲刺 |

### 4. 半马比赛策略

| 段落 | 配速建议 | 心率 |
|------|----------|------|
| 0-5K | 比目标慢 5-10 秒 | 75-80% 最大 |
| 5-10K | 目标配速 | 80-83% |
| 10-15K | 目标配速 | 83-85% |
| 15-18K | 目标或稍快 | 85-88% |
| 18-21K | 全力 | 88%+ |

### 5. 全马比赛策略

| 段落 | 配速建议 | 心率 |
|------|----------|------|
| 0-10K | 比目标慢 5-10 秒 | 75-78% |
| 10-21K | 目标配速 | 78-82% |
| 21-30K | 目标配速 | 82-85% |
| 30-35K | 最艰难，坚持 | 85-87% |
| 35-42K | 能跑多少跑多少 | 87%+ |

## 三、补给策略

### 1. 补给计划

**半马补给**：
```
赛前：早餐 + 跑前 15 分钟喝水
赛中：
- 每 5K 补水
- 45 分钟后能量胶 1 个
- 高温时盐丸
```

**全马补给**：
```
赛前：早餐 + 跑前 15 分钟喝水
赛中：
- 5K：水
- 10K：水 + 能量胶 1
- 15K：运动饮料
- 20K：水 + 能量胶 2 + 盐丸
- 25K：运动饮料
- 30K：水 + 能量胶 3 + 盐丸
- 35K：运动饮料 + 能量胶 4
- 40K：水
```

### 2. 补给站技巧

- 提前看地图，知道位置
- 不要跳过补给站
- 小口喝，不要大口灌
- 能量胶配水
- 后程站可能人多，可以往后站

## 四、心理策略

### 1. 比赛中分段目标

**避免想太多**：
- 不要想"还有 30K"
- 想当下、想下一个目标

**小目标法**：
```
"跑到下一个补给站"
"跑到那棵树"
"跑到下一个公里牌"
"只剩 5K 了"
```

### 2. 正面对话

**负面想法 → 正面对话**：
| 负面 | 正面 |
|------|------|
| "好累，跑不动了" | "这是正常的，我可以坚持" |
| "还有那么远" | "只剩 XK 了" |
| "腿好酸" | "我训练过，我可以" |
| "别人都比我快" | "按自己的节奏跑" |

### 3. 应对困难

**撞墙期（30-35K）**：
- 接受这是正常的
- 补充能量胶
- 缩小步伐，加快步频
- 专注下一个目标
- 想想训练的辛苦

**抽筋**：
- 停下拉伸
- 补充电解质
- 降低配速
- 不要硬撑

## 五、赛后恢复

### 1. 完赛后立即

- 不要立刻坐下
- 继续走动 10-15 分钟
- 补水
- 吃碳水+蛋白质
- 拿奖牌，拍照！

### 2. 赛后 24 小时

- 冷水浴或冰敷（可选）
- 轻柔拉伸
- 充足睡眠
- 继续补水

### 3. 赛后一周

| 天数 | 建议 |
|------|------|
| 1-3 天 | 休息或散步 |
| 4-7 天 | 轻松跑 20-30 分钟 |

### 4. 赛后调整

- 庆祝完成！
- 记录成绩和感受
- 分析什么做得好/不好
- 制定下一个目标

## 六、常见错误

| 错误 | 后果 | 正确做法 |
|------|------|----------|
| 起跑太快 | 后程崩溃 | 前程保守 |
| 不做减量 | 疲劳未消除 | 赛前 2-3 周减量 |
| 新装备比赛 | 磨损、不适 | 赛前磨合 50K+ |
| 跳过补给站 | 能量不足 | 按计划补给 |
| 不熟悉赛道 | 意外消耗 | 提前研究 |

## 七、常见问题

### Q1：比赛前应该减量多久？

**根据比赛距离决定。**

**减量周期参考**：

| 比赛距离 | 减量时间 |
|----------|----------|
| 5K | 3-5天 |
| 10K | 5-7天 |
| 半马 | 1-2周 |
| 全马 | 2-3周 |

**减量原则**：
- 跑量减少，强度保持
- 最后一周跑量约为正常的30-40%
- 不要完全停止训练

### Q2：比赛当天早餐应该吃什么？

**熟悉、易消化、足够碳水。**

**推荐早餐**：

| 食物 | 份量 | 时机 |
|------|------|------|
| 面包 + 香蕉 | 2片 + 1根 | 赛前3小时 |
| 燕麦 + 水果 | 1碗 | 赛前3小时 |
| 馒头 + 鸡蛋 | 1个 + 1个 | 赛前3小时 |

**避免**：
- 高纤维食物
- 高脂肪食物
- 新尝试的食物
- 太油腻

### Q3：比赛起跑太快怎么办？

**这是最常见的错误，需要提前预防。**

**预防方法**：

| 方法 | 说明 |
|------|------|
| 设定配速提醒 | 手表设定目标配速警报 |
| 找配速组 | 跟着官方配速员 |
| 自我提醒 | 比赛前提醒自己"前慢后快" |
| 看心率 | 控制在目标区间下限 |

**如果已经起跑太快**：
- 立即降速
- 不要试图"追赶"失去的时间
- 后程可能还会追回

### Q4：比赛中抽筋怎么办？

**立即处理，不要硬撑。**

**抽筋处理**：

| 部位 | 拉伸方法 |
|------|----------|
| 小腿 | 站立，脚跟着地，身体前倾 |
| 大腿前侧 | 站立，弯曲膝盖，手抓脚踝 |
| 大腿后侧 | 站立，腿伸直，身体前倾 |

**预防**：
- 补充电解质/盐丸
- 赛前充分热身
- 不要起跑太快

### Q5：比赛时遇到撞墙期怎么办？

**接受这是正常的，采取应对措施。**

**撞墙期应对**：

| 策略 | 方法 |
|------|------|
| 心理 | 接受不适，专注当下 |
| 补给 | 立即补充能量胶 |
| 配速 | 可以稍微降速 |
| 目标 | 设小目标（下一公里、下一个补给站）|
| 想法 | 想想训练的辛苦，相信自己 |

**撞墙期特点**：
- 通常发生在30-35公里
- 可能持续5-15分钟
- 补给后可能缓解

### Q6：比赛应该穿什么？

**穿经过测试的熟悉装备。**

**穿着原则**：

| 原则 | 说明 |
|------|------|
| 不穿新装备 | 至少磨合50公里 |
| 分层穿着 | 根据温度调整 |
| 注意摩擦点 | 贴防磨膏 |
| 温度+10度 | 按比实际温度高10度穿着 |

**温度参考**：

| 温度 | 穿着建议 |
|------|----------|
| < 10°C | 长袖 + 短裤或紧身裤 |
| 10-15°C | 短袖 + 短裤 |
| 15-20°C | 轻薄短袖 + 短裤 |
| > 20°C | 最轻薄装备 |

### Q7：比赛前一天能跑步吗？

**可以，但量要很少。**

**赛前一天建议**：

| 比赛距离 | 赛前一天 |
|----------|----------|
| 5K/10K | 轻松跑20分钟或不跑 |
| 半马 | 轻松跑20-30分钟 |
| 全马 | 轻松跑20-30分钟 + 2×100米加速 |

**目的**：保持身体"激活"，不让肌肉僵硬。

### Q8：比赛后多久可以恢复跑步？

**根据比赛距离和身体状态。**

**恢复时间参考**：

| 比赛距离 | 完全恢复跑步 |
|----------|--------------|
| 5K | 2-3天 |
| 10K | 3-5天 |
| 半马 | 7-14天 |
| 全马 | 14-28天 |

**恢复原则**：
- 没有疼痛再开始跑步
- 从轻松跑开始
- 逐渐增加跑量

### Q9：比赛应该带什么？

**根据比赛距离和补给安排。**

**携带物品清单**：

| 距离 | 必带 | 可选 |
|------|------|------|
| 5K | 无 | 无 |
| 10K | 无 | 能量胶1个 |
| 半马 | 能量胶2个 | 盐丸、腰包 |
| 全马 | 能量胶4-6个、盐丸 | 腰包/背心 |

**建议**：
- 熟悉的补给品
- 不要带太多
- 提前练习携带

### Q10：比赛中想上厕所怎么办？

**提前预防，赛中灵活处理。**

**预防方法**：
- 赛前排空
- 不要喝太多咖啡因
- 赛前早餐不要太油腻

**赛中处理**：
- 了解赛道厕所位置
- 不要不好意思
- 短暂停留影响不大

### Q11：下雨天比赛怎么办？

**调整装备和心态。**

**雨天比赛策略**：

| 方面 | 建议 |
|------|------|
| 装备 | 防水外套、帽子、旧鞋 |
| 热身 | 充分热身，保持温暖 |
| 赛道 | 注意积水、路滑 |
| 补给 | 注意保暖 |
| 心态 | 接受条件，专注自己 |

**注意**：
- 起跑区可能湿滑
- 转弯处减速
- 终点后尽快换干衣服

### Q12：首马应该定什么目标？

**完赛就是最好的目标。**

**首马建议**：

| 方面 | 建议 |
|------|------|
| 目标 | 完赛，不追求成绩 |
| 配速 | 保守，起跑要慢 |
| 补给 | 按计划，不要跳过 |
| 心态 | 享受比赛，这是一次体验 |

**不要**：
- 根据半马成绩简单翻倍
- 盲目追求"破4"等目标
- 被他人配速影响

**首马意义**：体验马拉松，学习比赛，为以后积累经验。
