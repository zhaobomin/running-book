# 营养恢复

> 燃料决定表现，恢复是训练的一部分

## 本节内容

营养和恢复是跑步训练中不可忽视的环节。正确的营养摄入和充足的恢复能让你训练更有效、表现更好。

## 营养基础

### 三大营养素

| 营养素 | 作用 | 占比 |
|--------|------|------|
| 碳水化合物 | 主要能量来源 | 50-60% |
| 蛋白质 | 肌肉修复 | 15-20% |
| 脂肪 | 长期能量、激素 | 20-30% |

### 水分需求

- 日常：体重(kg) × 30-35ml
- 跑前 2 小时：500ml
- 跑中：每 20 分钟 150-200ml
- 跑后：补充流失体重的 150%

## 恢复要点

### 睡眠
- 每晚 7-9 小时
- 固定作息时间
- 良好的睡眠环境

### 主动恢复
- 散步、游泳、骑车
- 瑜伽、拉伸
- 泡沫轴放松

### 恢复时间

| 训练类型 | 恢复时间 |
|----------|----------|
| 轻松跑 | 24 小时 |
| 节奏跑 | 24-48 小时 |
| 间歇跑 | 48-72 小时 |
| 长距离（30K+）| 72-96 小时 |
| 全马比赛 | 14-28 天 |

---

*营养和恢复是训练的基石。再好的训练计划，没有足够的营养和恢复支持，效果也会大打折扣。*
