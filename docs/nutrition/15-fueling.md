# 长距离补给策略

> 15K 以上的补给决定成败

## 一、为什么长距离需要补给？

### 1. 身体在长距离中的变化

| 时间 | 身体变化 |
|------|----------|
| 60 分钟 | 糖原开始耗尽 |
| 90 分钟 | 糖原明显不足 |
| 120 分钟 | 可能"撞墙" |
| 150+ 分钟 | 需要持续补给 |

### 2. 需要补给的内容

| 补给品 | 作用 | 何时需要 |
|--------|------|----------|
| 水 | 体温调节、代谢 | 始终 |
| 电解质 | 防止抽筋、保持体液平衡 | >60 分钟，高温 |
| 碳水化合物 | 维持血糖、能量 | >60 分钟 |
| 盐丸 | 补充钠、防止低钠血症 | 大量出汗时 |

## 二、补给计划

### 1. 根据距离制定

**10-15K**
```
补水：每 3-5K 喝水
碳水：通常不需要
电解质：高温时运动饮料
```

**15-21K（半马）**
```
补水：每 3-5K
碳水：45 分钟后开始，每 45 分钟 30g
电解质：运动饮料或盐丸
```

**30K+（全马训练）**
```
补水：每 3-5K
碳水：30 分钟后开始，每 45 分钟 30-60g
电解质：每小时盐丸 1-2 颗
```

**42K（全马比赛）**
```
补水：每站必喝
碳水：30 分钟开始，共 4-6 个能量胶
电解质：每小时盐丸
```

### 2. 根据时间制定

**< 60 分钟**
- 只需要水

**60-120 分钟**
- 每 45 分钟 30g 碳水
- 运动饮料或能量胶
- 每 20 分钟补水

**> 120 分钟**
- 每 45 分钟 30-60g 碳水
- 能量胶 + 能量棒/香蕉
- 盐丸每小时
- 运动饮料 + 水交替

## 三、补水策略

### 1. 跑前补水
```
跑前 2-3 小时：500ml
跑前 15-30 分钟：250ml
尿液颜色：淡黄色最佳
```

### 2. 跑中补水

**原则**：
- 每 20 分钟 150-200ml
- 小口多次，不要大口灌
- 口渴时立即喝

**补水站策略**：
| 距离 | 补给 |
|------|------|
| 5K | 水 |
| 10K | 水 + 运动饮料 |
| 15K | 水 + 能量胶 |
| 20K | 运动 + 盐丸 |
| ... | 循环 |

### 3. 跑后补水
```
体重流失 × 1.5 = 需补充水量
例：流失 1kg → 补充 1.5L
```

### 4. 过度补水的风险
- 低钠血症（血钠过低）
- 症状：头晕、恶心、头痛
- 预防：不要只喝白水，配合电解质

## 四、碳水补给

### 1. 能量胶

**优点**：
- 便携
- 快速吸收
- 精确剂量

**使用方法**：
- 每 45-60 分钟一个
- 配水服用
- 不要一次吃多个

**常见品牌**：
- GU
- PowerBar
- SIS
- Maurten

### 2. 能量棒

**优点**：
- 饱腹感
- 持续能量

**适合**：
- 超长距离（>30K）
- 训练中途

**缺点**：
- 可能口渴
- 消化较慢

### 3. 固体食物

**适合长距离/超马**：
- 香蕉
- 葡萄干
- 白面包
- 土豆
- 年糕

### 4. 碳水含量参考

| 食物 | 碳水含量 |
|------|----------|
| 能量胶 | 20-25g |
| 香蕉 | 25-30g |
| 能量棒 | 30-40g |
| 运动饮料 500ml | 30-35g |
| 葡萄干 30g | 22g |

## 五、电解质补给

### 1. 为什么需要电解质？

**钠的作用**：
- 保持体液平衡
- 防止抽筋
- 神经肌肉功能

**缺钠症状**：
- 抽筋
- 疲劳
- 头晕
- 恶心

### 2. 电解质来源

**运动饮料**：
- 方便
- 同时补碳水
- 选择等渗或高渗

**盐丸**：
- 便携
- 精确
- 大量出汗时必需

**天然来源**：
- 咸味零食
- 电解质粉

### 3. 盐丸使用

**何时需要**：
- 大量出汗（高温、高湿）
- 跑步 > 2 小时
- 容易抽筋的人

**用量**：
- 每小时 1-2 颗
- 配水服用
- 不要过量

## 六、补给训练

### 1. 训练中练习

**为什么重要**：
> 比赛时不要尝试新东西！

**训练内容**：
- 练习携带补给
- 练习跑步中进食
- 测试不同品牌
- 找到适合自己的

### 2. 补给计划制定

**比赛补给清单**：
```
赛前：
□ 跑前 2-3 小时早餐
□ 跑前 15 分钟水

赛中：
□ 能量胶 4-6 个
□ 盐丸 4-6 颗
□ 熟悉补给站位置

计划：
- 5K：水
- 10K：水 + 能量胶 1
- 15K：运动饮料
- 20K：水 + 能量胶 2 + 盐丸
- 25K：运动饮料
- 30K：水 + 能量胶 3 + 盐丸
- 35K：运动饮料 + 能量胶 4
- 40K：水
```

## 七、肠胃问题

### 1. 常见肠胃问题

| 问题 | 原因 | 解决 |
|------|------|------|
| 恶心 | 补给过多/过快 | 减慢补给速度 |
| 腹泻 | 高渗饮料/纤维 | 降低浓度，避免纤维 |
| 腹痛 | 空腹或太饱 | 调整进食时间 |

### 2. 预防肠胃问题

- 训练时练习补给
- 不要尝试新食物
- 补给小口多次
- 能量胶配水
- 比赛日早餐熟悉

### 3. 肠胃友好策略

**适合敏感肠胃**：
- 低纤维能量胶
- 多种糖源（葡萄糖+果糖）
- 小量多次
- 训练肠道适应

## 八、常见问题

### Q1：能量胶哪种好？

**选择适合自己的，没有最好的。**

**选择考虑因素**：

| 因素 | 说明 |
|------|------|
| 口味 | 选择喜欢的，训练时测试 |
| 稠度 | 有些较稀，有些较稠 |
| 咖啡因 | 有些含咖啡因，提神作用 |
| 成分 | 葡萄糖、果糖、麦芽糊精等 |
| 包装 | 是否方便携带和打开 |

**常见品牌特点**：

| 品牌 | 特点 |
|------|------|
| GU | 经典，稠度适中 |
| SIS | 较稀，容易吞咽 |
| Maurten | 胶囊状，无味 |
| PowerBar | 多种口味选择 |

**建议**：训练时测试多种品牌，找到适合自己的。

### Q2：能量胶一次吃多少？

**一般一个，每45-60分钟一个。**

**补给指南**：

| 跑步时长 | 能量胶数量 |
|----------|------------|
| 60-90分钟 | 0-1个 |
| 90-120分钟 | 1-2个 |
| 2-3小时 | 2-3个 |
| 3-4小时 | 3-4个 |
| 全马 | 4-6个 |

**注意**：
- 配水服用
- 分次补给，不要一次吃多个
- 每45-60分钟最多一个

### Q3：跑中吃不下东西怎么办？

**尝试不同的补给方式。**

**解决方案**：

| 问题 | 解决方法 |
|------|----------|
| 能量胶太甜 | 选择清淡口味，或用水稀释 |
| 吃不下固体 | 选择液体补给 |
| 肠胃不适 | 小口多次，降低浓度 |
| 完全吃不下 | 分多次少量尝试 |

**替代方案**：
- 运动饮料（液体补给）
- 能量糖/软糖
- 稀释的能量胶
- 少量多次补给

### Q4：不补给能跑完全马吗？

**可以，但不建议。**

**不补给的后果**：

| 情况 | 可能后果 |
|------|----------|
| 不补水 | 脱水、中暑风险 |
| 不补能量 | 可能"撞墙"、配速骤降 |
| 不补电解质 | 抽筋、低钠血症风险 |

**建议**：
- 至少补水
- 争取补给能量
- 根据自身情况调整

### Q5：盐丸什么时候吃？

**大量出汗、长时间跑步时需要。**

**需要盐丸的情况**：

| 情况 | 建议 |
|------|------|
| 跑步 < 90分钟 | 通常不需要 |
| 跑步 90-120分钟 | 可考虑 |
| 跑步 > 2小时 | 建议补充 |
| 高温高湿 | 必须补充 |
| 容易抽筋 | 建议补充 |
| 出汗量大 | 建议补充 |

**用量**：每小时1-2颗，配水服用。

### Q6：运动饮料怎么选？

**根据渗透压和需求选择。**

**运动饮料类型**：

| 类型 | 特点 | 适用情况 |
|------|------|----------|
| 等渗 | 与体液浓度相似 | 大多数情况 |
| 高渗 | 浓度高于体液 | 快速能量补充 |
| 低渗 | 浓度低于体液 | 快速补水 |

**选择建议**：
- 大多数跑步：等渗运动饮料
- 高温补水：低渗或等渗
- 能量需求高：高渗

### Q7：比赛中如何携带补给？

**多种携带方式。**

**携带方式**：

| 方式 | 优点 | 适合 |
|------|------|------|
| 腰包 | 容量大，稳定 | 长距离 |
| 跑步背心 | 容量大，可放水袋 | 超长距离 |
| 短裤口袋 | 方便，少负担 | 少量补给 |
| 手持水壶 | 方便取用 | 中等距离 |
| 别在号码布上 | 简单 | 少量能量胶 |

**建议**：训练时练习携带方式，比赛时使用熟悉的方式。

### Q8：补给站怎么利用？

**提前了解，合理利用。**

**补给站策略**：

| 建议 | 说明 |
|------|------|
| 提前了解位置 | 查看赛道图 |
| 不要错过 | 每站都补水 |
| 小口多次 | 不要一次灌太多 |
| 注意脚下 | 避免滑倒 |
| 运动饮料+水交替 | 平衡补给 |

**取水技巧**：
- 提前观察志愿者位置
- 伸手示意
- 捏扁杯口方便喝
- 不要停在路中间

### Q9：比赛中肠胃不适怎么办？

**调整补给策略。**

**肠胃问题应对**：

| 症状 | 可能原因 | 应对 |
|------|----------|------|
| 恶心 | 补给过快/过多 | 放慢补给速度 |
| 腹泻 | 高渗饮料/纤维 | 换低渗饮料 |
| 胃痛 | 空腹或太饱 | 调整进食时间 |
| 腹胀 | 气体 | 避免碳酸饮料 |

**预防**：
- 训练时练习补给
- 比赛日不尝试新东西
- 小口多次
- 能量胶配水

### Q10：跑后应该立刻吃东西吗？

**建议30分钟内补充。**

**跑后补给时机**：

| 时间 | 效果 |
|------|------|
| 30分钟内 | 最佳 |
| 30-60分钟 | 很好 |
| 1-2小时 | 可接受 |
| > 2小时 | 恢复效果降低 |

**简便选择**：
- 巧克力奶
- 香蕉 + 酸奶
- 能量棒
- 蛋白质奶昔

**正餐选择**：
- 米饭 + 鸡肉 + 蔬菜
- 意面 + 牛肉 + 沙拉

### Q11：空腹跑长距离可以吗？

**不建议超过90分钟的空腹跑。**

**风险**：

| 风险 | 说明 |
|------|------|
| 低血糖 | 头晕、无力 |
| 表现下降 | 能量不足 |
| 恢复变慢 | 糖原储备低 |
| 肌肉分解 | 可能消耗肌肉 |

**建议**：
- 轻松跑（<60分钟）：可空腹
- 中等强度（60-90分钟）：少量进食
- 长距离（>90分钟）：必须进食

**空腹跑前少量进食**：
- 半根香蕉
- 几口燕麦
- 少量运动饮料

### Q12：如何训练肠胃适应补给？

**循序渐进，逐渐增加。**

**训练方法**：

| 阶段 | 训练内容 |
|------|----------|
| 第1-2周 | 跑中喝水 |
| 第3-4周 | 加入运动饮料 |
| 第5-6周 | 尝试能量胶 |
| 第7-8周 | 模拟比赛补给 |

**关键点**：
- 从少量开始
- 逐渐增加补给量
- 测试不同品牌
- 找到适合自己的节奏
