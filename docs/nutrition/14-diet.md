# 跑者饮食指南

> 燃料决定表现

## 一、营养基础

### 1. 三大营养素

| 营养素 | 作用 | 占比 | 来源 |
|--------|------|------|------|
| 碳水化合物 | 主要能量来源 | 50-60% | 米饭、面食、燕麦、水果 |
| 蛋白质 | 肌肉修复 | 15-20% | 鸡肉、鱼、蛋、豆类 |
| 脂肪 | 长期能量、激素 | 20-30% | 坚果、橄榄油、牛油果 |

### 2. 微量营养素

**跑者重要的维生素和矿物质**：
| 营养素 | 作用 | 来源 |
|--------|------|------|
| 铁 | 氧气运输 | 红肉、菠菜、豆类 |
| 钙 | 骨骼健康 | 牛奶、酸奶、绿叶菜 |
| 维生素D | 骨骼、免疫 | 阳光、鱼、蛋黄 |
| 镁 | 肌肉功能 | 坚果、绿叶菜、香蕉 |
| 钾 | 电解质平衡 | 香蕉、土豆、橙子 |

### 3. 水分

**每日基础**：体重(kg) × 30-35ml

**跑步时**：
- 跑前 2 小时：500ml
- 跑中：每 20 分钟 150-200ml
- 跑后：补充流失体重的 150%

## 二、日常饮食指南

### 1. 训练日饮食

**早餐示例**：
```
燕麦 + 香蕉 + 坚果 + 牛奶
或
全麦面包 + 鸡蛋 + 牛油果
或
酸奶 + 水果 + 格兰诺拉麦片
```

**午餐示例**：
```
米饭 + 鸡胸肉 + 蔬菜
或
全麦意面 + 三文鱼 + 沙拉
或
红薯 + 牛肉 + 西兰花
```

**晚餐示例**：
```
糙米 + 鱼 + 烤蔬菜
或
藜麦 + 豆腐 + 炒菜
或
土豆 + 瘦肉 + 沙拉
```

### 2. 休息日饮食

- 碳水稍减少
- 蛋白质保持
- 多吃蔬菜水果
- 保持水分

### 3. 加餐选择

**健康加餐**：
- 香蕉 + 花生酱
- 希腊酸奶 + 浆果
- 坚果 + 干果
- 全麦饼干 + 奶酪
- 煮鸡蛋

## 三、跑前饮食

### 1. 时间指南

| 时间 | 食物量 | 例子 |
|------|--------|------|
| 3-4 小时前 | 正餐 | 米饭 + 鸡肉 + 蔬菜 |
| 2-3 小时前 | 轻餐 | 面包 + 香蕉 |
| 1 小时前 | 小吃 | 香蕉 / 能量棒 |
| 30 分钟前 | 极少量 | 几口运动饮料 |

### 2. 跑前食物选择

**适合吃**：
- 香蕉
- 燕麦
- 全麦面包
- 能量棒
- 酸奶

**避免吃**：
- 高纤维食物（可能引起肠胃不适）
- 高脂肪食物（消化慢）
- 辛辣食物
- 新食物

## 四、跑中补给

### 1. 什么时候需要？

| 跑步时长 | 是否需要补给 |
|----------|--------------|
| < 60 分钟 | 只需水 |
| 60-90 分钟 | 可能需要电解质 |
| > 90 分钟 | 需要碳水 + 电解质 |

### 2. 长距离补给

**碳水补给**：
- 每 45-60 分钟 30-60g 碳水
- 能量胶、能量棒、香蕉
- 小口多次

**电解质补给**：
- 运动饮料
- 盐丸（大量出汗时）
- 每 60 分钟

**补水**：
- 每 20 分钟 150-200ml
- 口渴时喝
- 不要过量

## 五、跑后恢复

### 1. 恢复窗口

**黄金时间**：跑后 30-60 分钟

**目标**：
- 补充糖原
- 肌肉修复
- 重新水合

### 2. 恢复食物

**碳水 + 蛋白质比例**：3:1 或 4:1

**便捷选择**：
- 巧克力奶
- 香蕉 + 酸奶
- 能量棒
- 三明治
- 蛋白质奶昔 + 香蕉

**正餐选择**：
- 米饭 + 鸡肉 + 蔬菜
- 意面 + 牛肉 + 沙拉
- 红薯 + 鱼 + 西兰花

## 六、特殊考虑

### 1. 减重期

**原则**：
- 热量缺口不要太大（300-500卡/天）
- 保持蛋白质摄入
- 不要在训练前后减少碳水
- 保证微量元素

### 2. 赛前 Loading

**赛前 2-3 天**：
- 碳水增加到每公斤体重 7-10g
- 减少脂肪和纤维
- 保持水分

### 3. 素食跑者

**注意补充**：
- 蛋白质（豆类、豆腐、乳制品）
- 铁（菠菜、豆类 + 维生素C）
- 维生素B12（补充剂）
- 钙（强化植物奶、豆腐）

### 4. 晨跑

**跑前**：
- 可以空腹跑（轻松跑，<60分钟）
- 或吃少量碳水（香蕉、几口燕麦）

**跑后**：
- 尽快吃早餐
- 碳水 + 蛋白质

## 七、常见问题

### Q1：跑前多久吃东西？

**根据跑步强度和时长决定。**

| 跑步类型 | 建议进食时间 | 食物类型 |
|----------|--------------|----------|
| 轻松跑（<60分钟）| 可空腹或30分钟前小吃 | 香蕉、几口燕麦 |
| 中等强度（60-90分钟）| 1-2小时前 | 轻餐 |
| 长距离/强度训练 | 2-3小时前 | 正餐 |
| 比赛 | 3小时前 | 熟悉的早餐 |

**原则**：跑步强度越高，需要消化的时间越长。

### Q2：跑后多久吃东西？

**最佳时机：30-60分钟内。**

**恢复窗口**：

| 时间 | 效果 |
|------|------|
| 30分钟内 | 最佳，肌肉对营养最敏感 |
| 30-60分钟 | 很好 |
| 60-120分钟 | 可接受 |
| > 2小时 | 恢复效果降低 |

**简便选择**：巧克力奶、香蕉+酸奶、蛋白质奶昔。

### Q3：需要补充蛋白粉吗？

**不一定。取决于日常饮食。**

**判断标准**：

| 情况 | 建议 |
|------|------|
| 日常蛋白质摄入足够 | 不需要 |
| 素食者/饮食受限 | 可能需要 |
| 高强度训练期 | 可考虑 |
| 减重期 | 有帮助 |

**蛋白质需求量**：每公斤体重 1.4-1.7g。

**示例**：60kg跑者每天需要 85-100g 蛋白质。

**天然蛋白质来源**：

| 食物 | 蛋白质含量 |
|------|------------|
| 鸡胸肉 100g | 31g |
| 牛肉 100g | 26g |
| 鱼 100g | 20-25g |
| 鸡蛋 1个 | 6g |
| 希腊酸奶 150g | 15g |
| 豆腐 100g | 8g |

### Q4：能量胶什么时候吃？

**跑步超过 90 分钟才需要。**

**使用指南**：

| 跑步时长 | 能量胶建议 |
|----------|------------|
| < 60 分钟 | 不需要 |
| 60-90 分钟 | 可能需要 1 个 |
| 90-120 分钟 | 1-2 个 |
| > 120 分钟 | 每 45-60 分钟 1 个 |

**注意事项**：
- 配水服用
- 不要一次吃多个
- 训练时测试过再比赛使用
- 选择适合自己口味的

### Q5：喝什么水最好？

**根据跑步时长和条件选择。**

**选择指南**：

| 情况 | 推荐 |
|------|------|
| 日常 | 普通水 |
| 轻松跑（<60分钟）| 普通水 |
| 长距离（>60分钟）| 运动饮料 |
| 高温大量出汗 | 运动饮料 + 盐丸 |
| 减重期 | 普通水或低糖运动饮料 |

**运动饮料选择**：
- 等渗：适合大多数情况
- 高渗：高强度时
- 低渗：快速补水

### Q6：跑步能减肥吗？

**能，但需要配合饮食控制。**

**减肥原理**：
- 热量消耗 > 热量摄入
- 跑步30分钟约消耗 200-400 卡

**常见误区**：

| 误区 | 正确做法 |
|------|----------|
| 跑完随便吃 | 控制总热量 |
| 只关注体重 | 关注体脂变化 |
| 不吃碳水 | 选择优质碳水 |
| 空腹跑更减肥 | 不一定，可能影响训练质量 |

**建议**：
- 热量缺口控制在 300-500 卡/天
- 保持蛋白质摄入
- 不要在训练前后节食

### Q7：跑步时应该喝多少水？

**根据出汗量和时长调整。**

**补水指南**：

| 时机 | 建议量 |
|------|--------|
| 跑前2小时 | 500ml |
| 跑前15-30分钟 | 250ml |
| 跑中每20分钟 | 150-200ml |
| 跑后 | 体重流失 × 1.5 |

**判断是否补水足够**：
- 尿液颜色：淡黄色
- 不感到口渴
- 体重变化 < 2%

### Q8：赛前碳水装载怎么吃？

**赛前2-3天开始增加碳水摄入。**

**碳水装载计划**：

| 时间 | 碳水摄入 |
|------|----------|
| 赛前3天 | 7-10g/kg体重 |
| 赛前2天 | 7-10g/kg体重 |
| 赛前1天 | 7-10g/kg体重 |
| 比赛日早晨 | 100-150g（3小时前）|

**示例**：60kg跑者，赛前每天需要 420-600g 碳水。

**注意事项**：
- 减少高纤维食物
- 避免新食物
- 保持水分充足
- 不要吃太油腻

### Q9：素食跑者需要注意什么？

**特别注意蛋白质、铁、B12的摄入。**

**素食跑者营养要点**：

| 营养素 | 植物来源 | 注意事项 |
|--------|----------|----------|
| 蛋白质 | 豆类、豆腐、藜麦 | 多样化搭配 |
| 铁 | 菠菜、豆类、坚果 | 搭配维生素C促进吸收 |
| 维生素B12 | 强化食品、补充剂 | 可能需要补充 |
| 钙 | 豆腐、强化植物奶 | 注意摄入量 |
| 锌 | 坚果、种子、全谷物 | - |

**建议**：定期检查血液指标，必要时补充。

### Q10：跑前可以喝咖啡吗？

**可以，有一定益处。**

**咖啡因的作用**：

| 作用 | 说明 |
|------|------|
| 提高警觉性 | 神经系统兴奋 |
| 延缓疲劳 | 可能提高耐力表现 |
| 促进脂肪利用 | 节省糖原 |

**建议摄入量**：
- 3-6mg/kg体重
- 跑前 30-60 分钟

**注意事项**：
- 不习惯咖啡因者要谨慎
- 可能影响睡眠
- 可能导致肠胃不适
- 不要过量

### Q11：跑步时要吃盐丸吗？

**视情况而定。**

**需要盐丸的情况**：

| 情况 | 建议 |
|------|------|
| 跑步 < 90 分钟 | 通常不需要 |
| 跑步 > 2 小时 | 建议补充 |
| 高温高湿 | 需要 |
| 大量出汗 | 需要 |
| 容易抽筋 | 需要 |

**用量**：每小时 1-2 颗，配水服用。

**盐丸的作用**：
- 补充钠
- 防止低钠血症
- 预防抽筋
- 保持体液平衡

### Q12：早餐应该吃什么？

**根据当天训练安排。**

**早餐选择**：

| 训练安排 | 早餐建议 |
|----------|----------|
| 休息日 | 正常早餐，可减少碳水 |
| 轻松跑日 | 适中碳水 |
| 长距离/强度日 | 高碳水早餐 |
| 比赛日 | 熟悉的、高碳水早餐 |

**推荐早餐组合**：
- 燕麦 + 香蕉 + 坚果 + 牛奶
- 全麦面包 + 鸡蛋 + 牛油果
- 酸奶 + 水果 + 格兰诺拉麦片
- 米饭 + 少量肉 + 蔬菜（东亚风格）

**避免**：
- 高纤维（跑前）
- 高脂肪
- 太油腻
- 新尝试的食物
