## 训练原则

### 1. 循序渐进
每周跑量增加不超过10%，给身体适应的时间。

### 2. 80/20法则
80%的训练在轻松配速下进行，只有20%需要较高强度。

### 3. 休息是训练的一部分
没有足够的恢复，训练效果无法体现。

### 4. 倾听身体
小问题及时处理，避免变成大伤病。
