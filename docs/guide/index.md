# 简介

欢迎来到《跑步指南》！

本书旨在帮助跑者建立科学的跑步习惯，无论你是刚入门的新手，还是希望提升成绩的进阶跑者，都能在这里找到有价值的内容。

::: tip 关于本书
本书内容参考了 Jack Daniels《Running Formula》、Pete Pfitzinger《Advanced Marathoning》等经典跑步训练书籍，结合运动科学研究成果编写而成。
:::

## 内容概览

本书涵盖六大核心模块，帮助你全面掌握跑步知识：

| 模块 | 内容简介 |
|------|----------|
| [训练计划](/training/) | 从零基础到马拉松的完整训练方案 |
| [跑步技术](/technique/) | 姿势、步频、呼吸等核心技术要点 |
| [力量训练](/strength/) | 核心、下肢力量及热身拉伸指南 |
| [伤病预防](/injury/) | 常见伤病识别、预防与康复策略 |
| [营养恢复](/nutrition/) | 饮食、补给与恢复的科学方法 |
| [进阶话题](/advanced/) | 速度训练、心率训练与比赛策略 |

## 学习路径

```
新手入门 ──→ 巩固基础 ──→ 进阶提升 ──→ 挑战自我
    │              │              │              │
 训练计划      跑步技术       力量训练       进阶话题
    │              │              │              │
 伤病预防  ←──  营养恢复  ←──  贯穿始终  ←──  全程关注
```

### 详细学习建议

| 阶段 | 重点内容 | 建议时长 |
|------|----------|----------|
| 新手入门 | 训练计划 → 跑步技术 → 伤病预防 | 2-3 个月 |
| 巩固基础 | 力量训练 → 营养恢复 | 持续进行 |
| 进阶提升 | 心率训练 → 速度训练 | 3-6 个月 |
| 挑战自我 | 比赛策略 → 长距离补给 | 备赛期 |

## 适合人群

- **零基础新手** — 从未跑步，想要开始
- **初级跑者** — 能跑 3-5 公里，想要提升
- **进阶跑者** — 有一定基础，目标半马/全马
- **资深跑者** — 追求 PB，优化训练方法

## 如何使用本书

1. **新手**：从 [快速开始](/guide/getting-started) 了解基础准备，然后进入 [新手训练计划](/training/01-beginner)
2. **有经验跑者**：直接跳转到感兴趣的模块，或查看 [进阶话题](/advanced/)
3. **遇到问题**：查阅 [伤病预防](/injury/) 或 [营养恢复](/nutrition/) 相关章节
