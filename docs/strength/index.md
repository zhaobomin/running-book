# 力量训练

> 强壮的身体是跑步的基础

## 本节内容

力量训练是跑步训练的重要组成部分。它能提高跑步效率、预防伤病、让你在后程保持配速。

## 为什么要练力量？

| 好处 | 说明 |
|------|------|
| 提高效率 | 每步更省力，能量消耗更少 |
| 预防伤病 | 关节更稳定，冲击力更好地被吸收 |
| 改善跑姿 | 维持良好姿势的能力更强 |
| 后程表现 | 肌肉不疲劳，最后几公里保持配速 |

## 训练安排

### 每周力量训练建议

| 水平 | 频率 | 时间 |
|------|------|------|
| 新手 | 每周 2 次 | 每次 15-20 分钟 |
| 进阶 | 每周 2-3 次 | 每次 20-30 分钟 |
| 高手 | 每周 3 次 | 每次 30-40 分钟 |

### 训练时机

- **跑后**：10-15 分钟核心或下肢训练
- **休息日**：单独进行更完整的训练
- **跑前**：只做轻量激活（2-3分钟）

---

*将力量训练视为跑步训练的一部分，每周只需 2-3 次，每次 20-30 分钟，几周后你就会感受到明显的不同。*
