## 二、训练负荷管理

### 1. 10% 原则
每周跑量增加不超过 10%

**示例**：
| 周次 | 周跑量 | 增加量 |
|------|--------|--------|
| 第 1 周 | 20K | - |
| 第 2 周 | 22K | +10% |
| 第 3 周 | 24K | +9% |
| 第 4 周 | 20K | 减量周 |
| 第 5 周 | 26K | +8% |

### 2. 减量周
每 3-4 周安排一个减量周：
- 跑量减少 20-30%
- 让身体恢复
- 预防过度训练

### 3. 训练强度分配
```
80/20 原则：
- 80% 轻松跑
- 20% 高强度训练

很多跑者的问题是：
- 轻松跑不够轻松
- 高强度不够高
- 导致中间状态，效率低且受伤风险高
```

### 4. 监控训练负荷

**跑量记录**：
- 每周总距离
- 每周总时间
- 长距离距离

**疲劳信号**：
- 静息心率升高 5+ 次
- 睡眠质量下降
- 情绪烦躁
- 表现下降
- 频繁小伤病

## 三、恢复与休息

### 1. 休息日安排
| 跑步频率 | 适合人群 | 休息日 |
|----------|----------|--------|
| 3 天/周 | 新手 | 4 天 |
| 4-5 天/周 | 进阶 | 2-3 天 |
| 6 天/周 | 高水平 | 1 天 |
| 7 天/周 | 不推荐 | 0 天 |

### 2. 睡眠
**推荐**：每晚 7-9 小时

**睡眠的作用**：
- 肌肉修复
- 激素平衡
- 免疫力提升
- 心理恢复

**提高睡眠质量**：
- 固定作息
- 睡前 1 小时不用手机
- 保持卧室凉爽黑暗
- 避免睡前剧烈运动

### 3. 主动恢复

**比完全休息更好**：
- 轻松散步
- 游泳
- 骑车
- 瑜伽
- 泡沫轴放松

### 4. 营养恢复

**跑后 30-60 分钟内**：
- 碳水：蛋白质 = 3:1 或 4:1
- 如：香蕉 + 酸奶、巧克力奶

**日常营养**：
- 充足蛋白质（每公斤 1.4-1.7g）
- 均衡饮食
- 充足水分

## 四、力量与柔韧性

### 1. 力量训练

**每周 2-3 次，重点**：
- 臀中肌（稳定骨盆）
- 臀大肌（推蹬力量）
- 核心（躯干稳定）
- 小腿和脚踝（稳定落地）

**必做动作**：
1. 单腿站立（脚踝稳定）
2. 臀桥/单腿臀桥（臀肌）
3. 蚌式开合（臀中肌）
4. 平板支撑（核心）
5. 单腿硬拉（腘绳肌+臀肌）

### 2. 柔韧性训练

**每天可以做的拉伸**：
- 小腿拉伸
- 髋屈肌拉伸
- 腘绳肌拉伸
- 臀肌拉伸

**泡沫轴放松**：
- 大腿前侧
- 大腿后侧
- 小腿
- 髂胫束

### 3. 活动度

**髋关节活动度**：
- 90/90 拉伸
- 髋绕环
- 弓步转体

**脚踝活动度**：
- 脚踝绕环
- 小腿拉伸
- 足底筋膜放松

## 五、技术与装备

### 1. 跑姿要点

**预防伤病的关键**：
- 落地点在重心下方
- 步频 170-180
- 身体微前倾（从脚踝）
- 不跨步过大

### 2. 跑鞋选择

**更换周期**：500-800 公里

**选择原则**：
- 根据脚型选择
- 试穿时比日常鞋大半码
- 下午试鞋（脚会肿胀）
- 考虑跑步风格

**跑鞋类型**：
| 类型 | 适合人群 |
|------|----------|
| 缓震型 | 体重较大、足弓正常 |
| 稳定型 | 轻微过度内旋 |
| 控制型 | 严重过度内旋 |
| 赤足/极简 | 有经验的跑者 |

### 3. 训练场地

**场地硬度排序**（从软到硬）：
1. 草地
2. 森林步道
3. 橡胶跑道
4. 沥青
5. 水泥

**建议**：
- 大部分在较软地面
- 偶尔在硬地面（适应比赛）
- 下坡注意控制

## 六、自我监测

### 1. 晨起心率
- 每天早晨测量静息心率
- 如果比平时高 5+ 次，可能过度训练
- 考虑减量或休息

### 2. 主观疲劳度
```
1-10 分制：
1-3：非常轻松
4-6：中等
7-8：困难
9-10：非常困难

大部分训练应该在 4-6 分
```

### 3. 身体信号记录

**记录内容**：
- 睡眠质量
- 肌肉酸痛程度
- 精神状态
- 小伤痛

**每周回顾**：
- 是否持续疲劳
- 是否有新疼痛
- 是否需要调整

### 一周预防性安排
```
周一：跑步 + 跑后拉伸
周二：力量训练 + 泡沫轴
周三：跑步 + 跑后拉伸

周五：力量训练 + 拉伸
周六：长距离 + 跑后拉伸
周日：完全休息或主动恢复
```

## 九、常见问题

### Q1：每周跑量增加10%是硬性规定吗？

**不是，而是一个安全上限。**

**调整建议**：

| 情况 | 建议增幅 |
|------|----------|
| 新手 | 每周增加 5-10% |
| 有基础 | 每周增加 5-10% |
| 恢复期 | 每周增加 < 5% |
| 高水平跑者 | 可灵活调整 |

**原则**：宁可保守，不要冒险。身体适应需要时间。

### Q2：休息日是躺着不动吗？

**不是，主动恢复比完全休息更好。**

**休息日活动建议**：

| 活动类型 | 强度 | 时长 |
|----------|------|------|
| 散步 | 非常轻松 | 20-30分钟 |
| 游泳 | 轻松 | 20-30分钟 |
| 瑜伽 | 轻松 | 30-45分钟 |
| 泡沫轴放松 | - | 10-15分钟 |
| 拉伸 | - | 10-15分钟 |

**避免**：高强度训练、长时间站立。

### Q3：力量训练应该放在跑步前还是后？

**建议跑后或单独训练日。**

**安排建议**：

| 方式 | 适用情况 | 说明 |
|------|----------|------|
| 跑后 | 大多数情况 | 跑步是主要训练 |
| 单独训练日 | 备赛期、力量弱 | 更充分的力量训练 |
| 跑前 | 不推荐 | 可能影响跑步质量 |

**例外**：跑前可做2-3分钟的激活训练（如臀桥、深蹲各10次）。

### Q4：跑步时可以带伤痛跑吗？

**需要根据疼痛程度判断。**

**疼痛等级与建议**：

| 疼痛等级 | 描述 | 建议 |
|----------|------|------|
| 0 | 无痛 | 正常训练 |
| 1-2 | 轻微不适 | 可继续，注意观察 |
| 3 | 轻度疼痛 | 谨慎继续 |
| 4-5 | 中度疼痛 | 停止跑步 |
| 6+ | 明显疼痛 | 完全休息 |

**注意**：疼痛影响跑姿时，必须停止。

### Q5：跑鞋真的需要每500-800公里换吗？

**是的，但也要看鞋的状态。**

**判断跑鞋是否需要更换**：

| 指标 | 需要更换 |
|------|----------|
| 鞋底磨损 | 中底明显压缩、外底磨损 |
| 鞋变形 | 鞋后跟倾斜 |
| 缓震下降 | 跑时感觉更硬 |
| 跑后不适 | 脚、膝盖、小腿不适增加 |
| 里程 | > 800公里 |

**建议**：记录跑鞋里程，定期检查状态。

### Q6：新手应该选缓震型还是稳定型跑鞋？

**根据足型选择。**

**足型与跑鞋对应**：

| 足型 | 跑步姿态 | 推荐鞋型 |
|------|----------|----------|
| 高足弓 | 外旋（内翻）| 缓震型 |
| 正常足弓 | 正常 | 缓震型或稳定型 |
| 低足弓 | 过度内旋 | 稳定型 |
| 扁平足 | 严重过度内旋 | 控制型 |

**建议**：去专业店铺做步态分析。

### Q7：晨跑好还是夜跑好？

**各有优缺点，选择适合自己的。**

**对比**：

| 时段 | 优点 | 缺点 |
|------|------|------|
| 晨跑 | 养成习惯，不受干扰 | 身体僵硬，需更多热身 |
| 夜跑 | 身体状态好，放松身心 | 可能影响睡眠，安全考虑 |

**关键**：固定时间更容易养成习惯，选择能坚持的时间。

### Q8：跑步后拉伸多久合适？

**建议 10-15 分钟。**

**拉伸时间建议**：

| 时长 | 效果 |
|------|------|
| < 5 分钟 | 基本放松，比不做强 |
| 5-10 分钟 | 适度放松 |
| 10-15 分钟 | 理想效果 |
| > 15 分钟 | 边际收益递减 |

**必拉伸部位**：
- 小腿
- 大腿后侧
- 大腿前侧
- 臀部
- 髋屈肌

### Q9：如何知道自己的跑姿是否正确？

**自我检查方法**：

| 检查项 | 正确 | 错误 |
|--------|------|------|
| 头部 | 目光平视 | 低头或仰头 |
| 肩膀 | 放松下沉 | 耸肩 |
| 身体 | 微前倾（从脚踝）| 弯腰或后仰 |
| 落地 | 重心下方 | 身体前方（跨步过大）|
| 步频 | 170-180 | < 160 |
| 声音 | 轻柔 | "啪啪"砸地声 |

**建议**：请人帮忙拍摄跑步视频，或使用跑步机前的镜子观察。

### Q10：疲劳训练会增加伤病风险吗？

**会，疲劳时跑姿容易变形。**

**疲劳跑的风险**：

| 风险 | 原因 |
|------|------|
| 跑姿变形 | 肌肉无力维持正确姿势 |
| 落地过重 | 肌肉缓震能力下降 |
| 反应变慢 | 神经系统疲劳 |
| 代偿动作 | 身体自动调整以避免疼痛 |

**建议**：
- 感觉疲劳时降低配速或减少距离
- 保证恢复再进行高强度训练

### Q11：力量训练真的能预防伤病吗？

**是的，研究证实有效。**

**研究数据**：
- 规律力量训练可降低跑步伤病风险 30-50%
- 臀中肌无力与跑者膝高度相关
- 核心力量不足与下背痛相关

**预防性力量训练重点**：

| 肌肉群 | 预防的伤病 |
|--------|------------|
| 臀中肌 | 跑者膝、髂胫束综合征 |
| 臀大肌 | 髋部痛、腘绳肌拉伤 |
| 股四头肌 | 跑者膝 |
| 核心 | 下背痛 |
| 小腿和脚踝 | 跟腱炎、足底筋膜炎 |

### Q12：新手应该注意哪些伤病预防要点？

**新手最易犯的错误**：

| 错误 | 后果 | 预防方法 |
|------|------|----------|
| 跑太快 | 疲劳、放弃 | 保持"能说话"配速 |
| 跑太多 | 过度训练、伤病 | 每周增加 < 10% |
| 不热身 | 肌肉拉伤 | 跑前5分钟动态热身 |
| 鞋子不合适 | 足部问题 | 专业店铺选购 |
| 忽视休息 | 恢复不足 | 每周至少2天休息 |
| 不拉伸 | 肌肉紧张 | 跑后拉伸10分钟 |

**最关键一点**：倾听身体，有不适就调整。
