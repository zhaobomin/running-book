# 跑者饮食指南

> 燃料决定表现

## 一、营养基础

### 1. 三大营养素

| 营养素 | 作用 | 占比 | 来源 |
|--------|------|------|------|
| 碳水化合物 | 主要能量来源 | 50-60% | 米饭、面食、燕麦、水果 |
| 蛋白质 | 肌肉修复 | 15-20% | 鸡肉、鱼、蛋、豆类 |
| 脂肪 | 长期能量、激素 | 20-30% | 坚果、橄榄油、牛油果 |

### 2. 微量营养素

**跑者重要的维生素和矿物质**：
| 营养素 | 作用 | 来源 |
|--------|------|------|
| 铁 | 氧气运输 | 红肉、菠菜、豆类 |
| 钙 | 骨骼健康 | 牛奶、酸奶、绿叶菜 |
| 维生素D | 骨骼、免疫 | 阳光、鱼、蛋黄 |
| 镁 | 肌肉功能 | 坚果、绿叶菜、香蕉 |
| 钾 | 电解质平衡 | 香蕉、土豆、橙子 |

### 3. 水分

**每日基础**：体重(kg) × 30-35ml

**跑步时**：
- 跑前 2 小时：500ml
- 跑中：每 20 分钟 150-200ml
- 跑后：补充流失体重的 150%

## 二、日常饮食指南

### 1. 训练日饮食

**早餐示例**：
```
燕麦 + 香蕉 + 坚果 + 牛奶
或
全麦面包 + 鸡蛋 + 牛油果
或
酸奶 + 水果 + 格兰诺拉麦片
```

**午餐示例**：
```
米饭 + 鸡胸肉 + 蔬菜
或
全麦意面 + 三文鱼 + 沙拉
或
红薯 + 牛肉 + 西兰花
```

**晚餐示例**：
```
糙米 + 鱼 + 烤蔬菜
或
藜麦 + 豆腐 + 炒菜
或
土豆 + 瘦肉 + 沙拉
```

### 2. 休息日饮食

- 碳水稍减少
- 蛋白质保持
- 多吃蔬菜水果
- 保持水分

### 3. 加餐选择

**健康加餐**：
- 香蕉 + 花生酱
- 希腊酸奶 + 浆果
- 坚果 + 干果
- 全麦饼干 + 奶酪
- 煮鸡蛋

## 三、跑前饮食

### 1. 时间指南

| 时间 | 食物量 | 例子 |
|------|--------|------|
| 3-4 小时前 | 正餐 | 米饭 + 鸡肉 + 蔬菜 |
| 2-3 小时前 | 轻餐 | 面包 + 香蕉 |
| 1 小时前 | 小吃 | 香蕉 / 能量棒 |
| 30 分钟前 | 极少量 | 几口运动饮料 |

### 2. 跑前食物选择

**适合吃**：
- 香蕉
- 燕麦
- 全麦面包
- 能量棒
- 酸奶

**避免吃**：
- 高纤维食物（可能引起肠胃不适）
- 高脂肪食物（消化慢）
- 辛辣食物
- 新食物

## 四、跑中补给

### 1. 什么时候需要？

| 跑步时长 | 是否需要补给 |
|----------|--------------|
| < 60 分钟 | 只需水 |
| 60-90 分钟 | 可能需要电解质 |
| > 90 分钟 | 需要碳水 + 电解质 |

### 2. 长距离补给

**碳水补给**：
- 每 45-60 分钟 30-60g 碳水
- 能量胶、能量棒、香蕉
- 小口多次

**电解质补给**：
- 运动饮料
- 盐丸（大量出汗时）
- 每 60 分钟

**补水**：
- 每 20 分钟 150-200ml
- 口渴时喝
- 不要过量

## 五、跑后恢复

### 1. 恢复窗口

**黄金时间**：跑后 30-60 分钟

**目标**：
- 补充糖原
- 肌肉修复
- 重新水合

### 2. 恢复食物

**碳水 + 蛋白质比例**：3:1 或 4:1

**便捷选择**：
- 巧克力奶
- 香蕉 + 酸奶
- 能量棒
- 三明治
- 蛋白质奶昔 + 香蕉

**正餐选择**：
- 米饭 + 鸡肉 + 蔬菜
- 意面 + 牛肉 + 沙拉
- 红薯 + 鱼 + 西兰花

## 六、特殊考虑

### 1. 减重期

**原则**：
- 热量缺口不要太大（300-500卡/天）
- 保持蛋白质摄入
- 不要在训练前后减少碳水
- 保证微量元素

### 2. 赛前 Loading

**赛前 2-3 天**：
- 碳水增加到每公斤体重 7-10g
- 减少脂肪和纤维
- 保持水分

### 3. 素食跑者

**注意补充**：
- 蛋白质（豆类、豆腐、乳制品）
- 铁（菠菜、豆类 + 维生素C）
- 维生素B12（补充剂）
- 钙（强化植物奶、豆腐）

### 4. 晨跑

**跑前**：
- 可以空腹跑（轻松跑，<60分钟）
- 或吃少量碳水（香蕉、几口燕麦）

**跑后**：
- 尽快吃早餐
- 碳水 + 蛋白质

## 七、常见问题

### Q1: 跑前多久吃东西？
轻松跑：可以空腹或 30-60 分钟前小吃
长距离/强度训练：2-3 小时前正餐

### Q2: 跑后多久吃东西？
理想是 30-60 分钟内
最晚 2 小时内

### Q3: 需要补充蛋白粉吗？
不一定。如果日常饮食蛋白质足够，不需要。
每公斤体重 1.4-1.7g 蛋白质为参考。

### Q4: 能量胶什么时候吃？
跑步超过 90 分钟才需要
每 45-60 分钟一个

### Q5: 喝什么水？
日常：普通水
长时间跑步：运动饮料（电解质）
高温大量出汗：加盐的运动饮料
