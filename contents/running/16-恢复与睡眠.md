# 恢复与睡眠

> 恢复是训练的一部分

## 一、恢复的重要性

### 1. 恢复的作用
```
训练 → 疲劳 → 恢复 → 超量恢复 → 表现提升
```

**没有足够的恢复**：
- 疲劳积累
- 表现下降
- 伤病风险增加
- 过度训练综合征

### 2. 恢复的组成部分

| 组成部分 | 占比 |
|----------|------|
| 睡眠 | 60% |
| 营养 | 20% |
| 主动恢复 | 10% |
| 其他（按摩、拉伸等）| 10% |

## 二、睡眠

### 1. 为什么睡眠最重要？

**睡眠期间发生**：
- 生长激素分泌（肌肉修复）
- 细胞再生
- 记忆巩固
- 免疫系统恢复

### 2. 睡眠需求

| 人群 | 推荐睡眠 |
|------|----------|
| 普通人 | 7-8 小时 |
| 跑者 | 7-9 小时 |
| 高强度训练期 | 8-10 小时 |

### 3. 提高睡眠质量

**睡眠环境**：
- 温度：18-22°C
- 黑暗：使用窗帘或眼罩
- 安静：耳塞或白噪音
- 舒适：适合的床垫和枕头

**睡前习惯**：
- 固定睡眠时间
- 睡前 1 小时不用电子设备
- 避免咖啡因（下午 2 点后）
- 避免酒精
- 放松活动（阅读、拉伸）

**睡眠仪式**：
```
睡前 60 分钟：
1. 关闭电子设备
2. 轻柔拉伸 5-10 分钟
3. 洗温水澡
4. 阅读或冥想
5. 就寝
```

### 4. 睡眠不足的影响

| 缺乏程度 | 影响 |
|----------|------|
| 1 小时 | 轻微影响表现 |
| 2 小时 | 恢复受损、伤病风险增加 |
| 3+ 小时 | 严重影响训练、免疫力下降 |

### 5. 午睡

**好处**：
- 补充睡眠不足
- 提高警觉性
- 促进恢复

**建议**：
- 时间：20-30 分钟
- 时间点：下午 1-3 点
- 不要太晚（影响晚上睡眠）

## 三、主动恢复

### 1. 什么是主动恢复？

**不是完全休息**，而是轻度活动促进血液循环和恢复。

### 2. 主动恢复活动

| 活动 | 强度 | 时间 |
|------|------|------|
| 散步 | 极轻 | 20-30 分钟 |
| 游泳 | 轻松 | 20-30 分钟 |
| 骑车 | 轻松 | 20-30 分钟 |
| 瑜伽 | 轻柔 | 30-45 分钟 |
| 拉伸 | - | 15-20 分钟 |
| 泡沫轴 | - | 10-15 分钟 |

### 3. 休息日安排

**完全休息 vs 主动恢复**：
| 情况 | 建议 |
|------|------|
| 激烈训练后 | 主动恢复 |
| 疲劳但无痛 | 主动恢复 |
| 有小伤痛 | 完全休息 |
| 生病 | 完全休息 |
| 精神疲劳 | 完全休息 |

## 四、其他恢复方法

### 1. 按摩

**好处**：
- 放松肌肉
- 促进血液循环
- 减少酸痛

**频率**：每周 1 次

**替代**：泡沫轴自我按摩

### 2. 冷热疗法

**冰浴/冷水浴**：
- 温度：10-15°C
- 时间：10-15 分钟
- 适合：高强度训练后

**热水浴/桑拿**：
- 放松肌肉
- 促进血液循环
- 时间：15-20 分钟

**冷热交替**：
- 1 分钟冷水 + 3 分钟热水
- 重复 3-4 次
- 结束于冷水

### 3. 压缩装备

**压缩袜/腿套**：
- 促进血液回流
- 减少肌肉震动
- 跑中和跑后都可穿

### 4. 抬腿

**方法**：
- 平躺，腿靠墙
- 10-15 分钟
- 促进血液回流

## 五、恢复监测

### 1. 主观感受

**每日问自己**：
- 睡眠质量如何？
- 肌肉是否酸痛？
- 精神状态如何？
- 是否有动力训练？

**1-10 评分**：
- < 5：需要更多恢复
- 5-7：正常
- > 7：状态良好

### 2. 客观指标

**晨起心率**：
- 每天早晨测量
- 比平时高 5+ 次：可能过度训练
- 需要减量或休息

**心率变异性（HRV）**：
- 越高说明恢复越好
- 可用智能手表/APP 测量

**体重**：
- 突然下降：可能脱水或恢复不足
- 突然上升：可能水分滞留

### 3. 过度训练信号

**警示信号**：
- 持续疲劳
- 表现下降
- 睡眠问题
- 情绪波动
- 频繁生病
- 静息心率升高
- 肌肉持续酸痛

**处理**：
- 减少训练量 50%
- 增加休息日
- 关注睡眠和营养
- 如持续，寻求专业帮助

## 六、恢复时间参考

### 1. 不同训练后的恢复

| 训练类型 | 恢复时间 |
|----------|----------|
| 轻松跑 | 24 小时 |
| 节奏跑 | 24-48 小时 |
| 间歇跑 | 48-72 小时 |
| 长距离（15-20K）| 48-72 小时 |
| 长距离（30K+）| 72-96 小时 |
| 比赛（半马）| 7-14 天 |
| 比赛（全马）| 14-28 天 |

### 2. 全马赛后恢复

**赛后当天**：
- 补水、补充营养
- 轻柔活动
- 休息

**第 1-3 天**：
- 散步或轻柔交叉训练
- 不要跑步

**第 4-7 天**：
- 轻松跑（如感觉良好）
- 每次 20-30 分钟

**第 2 周**：
- 恢复正常跑量 50%

**第 3-4 周**：
- 逐渐恢复
