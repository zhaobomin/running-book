# 比赛策略

> 训练是准备，比赛是艺术

## 一、赛前准备

### 1. 赛前一月

**训练调整**：
- 跑量逐渐减少（减量期）
- 保持强度，减少量
- 长距离减少但不停

**减量计划**：
| 时间 | 跑量 |
|------|------|
| 赛前 3 周 | 80% |
| 赛前 2 周 | 60% |
| 赛前 1 周 | 40% |

### 2. 赛前一周

**训练**：
- 周一：休息
- 周二：轻松跑 6K
- 周三：轻松跑 5K + 2×400m 比赛配速
- 周四：休息
- 周五：轻松跑 4K
- 周六：比赛日

**饮食**：
- 碳水 loading（每公斤体重 7-10g 碳水）
- 减少纤维
- 多喝水
- 避免新食物

**睡眠**：
- 重点在赛前 2-3 晚
- 比赛前一晚紧张很正常
- 重要的是前几晚睡好

### 3. 比赛前一天

**准备清单**：
- [ ] 号码布别好
- [ ] 芯片绑好
- [ ] 比赛装备准备好
- [ ] 能量胶、盐丸准备
- [ ] 熟悉赛道和补给站
- [ ] 设定配速提醒
- [ ] 早餐食物准备

**活动**：
- 轻松活动，不要过度
- 领取参赛包
- 熟悉起点位置
- 早睡

### 4. 比赛日早晨

**时间安排**：
| 时间 | 活动 |
|------|------|
| 赛前 3 小时 | 起床、早餐 |
| 赛前 2 小时 | 轻微活动、喝水 |
| 赛前 1 小时 | 到达起点 |
| 赛前 30 分钟 | 热身、存包 |
| 赛前 10 分钟 | 进入起跑区 |

**早餐建议**：
- 碳水 100-150g
- 低纤维、低脂肪
- 熟悉的食物
- 如：面包+香蕉+运动饮料

## 二、比赛配速策略

### 1. 通用原则

**负分割**（后半程更快）：
- 最推荐的策略
- 前半程保守
- 后半程保持或加速

**原因**：
- 起跑太快要付出代价
- 后程崩溃比前程稍慢更痛苦
- 心理上更轻松

### 2. 5K 比赛策略

| 段落 | 配速建议 | 心态 |
|------|----------|------|
| 0-1K | 比目标慢 5-10 秒 | 压住冲动 |
| 1-3K | 目标配速 | 找到节奏 |
| 3-4K | 保持 | 坚持住 |
| 4-5K | 全力 | 冲刺 |

### 3. 10K 比赛策略

| 段落 | 配速建议 | 心态 |
|------|----------|------|
| 0-2K | 比目标慢 5 秒 | 稳定开始 |
| 2-5K | 目标配速 | 保持节奏 |
| 5-8K | 目标配速 | 最艰难阶段 |
| 8-10K | 全力 | 冲刺 |

### 4. 半马比赛策略

| 段落 | 配速建议 | 心率区间 |
|------|----------|----------|
| 0-5K | 比目标慢 5-10 秒 | 最大心率的 75-80% |
| 5-10K | 目标配速 | 最大心率的 80-83% |
| 10-15K | 目标配速 | 最大心率的 83-85% |
| 15-18K | 目标或稍快 | 最大心率的 85-88% |
| 18-21K | 全力 | 最大心率的 88%+ |

### 5. 全马比赛策略

| 段落 | 配速建议 | 心率区间 |
|------|----------|----------|
| 0-10K | 比目标慢 5-10 秒 | 最大心率的 75-78% |
| 10-21K | 目标配速 | 最大心率的 78-82% |
| 21-30K | 目标配速 | 最大心率的 82-85% |
| 30-35K | 最艰难，坚持 | 最大心率的 85-87% |
| 35-42K | 能跑多少跑多少 | 最大心率的 87%+ |

## 三、补给策略

### 1. 补给计划

**半马补给**：
```
赛前：早餐 + 跑前 15 分钟喝水
赛中：
- 每 5K 补水
- 45 分钟后能量胶 1 个
- 高温时盐丸
```

**全马补给**：
```
赛前：早餐 + 跑前 15 分钟喝水
赛中：
- 5K：水
- 10K：水 + 能量胶 1
- 15K：运动饮料
- 20K：水 + 能量胶 2 + 盐丸
- 25K：运动饮料
- 30K：水 + 能量胶 3 + 盐丸
- 35K：运动饮料 + 能量胶 4
- 40K：水
```

### 2. 补给站技巧

- 提前看地图，知道位置
- 不要跳过补给站
- 小口喝，不要大口灌
- 能量胶配水
- 后程站可能人多，可以往后站

## 四、心理策略

### 1. 比赛中分段目标

**避免想太多**：
- 不要想"还有 30K"
- 想当下、想下一个目标

**小目标法**：
```
"跑到下一个补给站"
"跑到那棵树"
"跑到下一个公里牌"
"只剩 5K 了"
```

### 2. 正面对话

**负面想法 → 正面对话**：
| 负面 | 正面 |
|------|------|
| "好累，跑不动了" | "这是正常的，我可以坚持" |
| "还有那么远" | "只剩 XK 了" |
| "腿好酸" | "我训练过，我可以" |
| "别人都比我快" | "按自己的节奏跑" |

### 3. 应对困难

**撞墙期（30-35K）**：
- 接受这是正常的
- 补充能量胶
- 缩小步伐，加快步频
- 专注下一个目标
- 想想训练的辛苦

**抽筋**：
- 停下拉伸
- 补充电解质
- 降低配速
- 不要硬撑

## 五、赛后恢复

### 1. 完赛后立即

- 不要立刻坐下
- 继续走动 10-15 分钟
- 补水
- 吃碳水+蛋白质
- 拿奖牌，拍照！

### 2. 赛后 24 小时

- 冷水浴或冰敷（可选）
- 轻柔拉伸
- 充足睡眠
- 继续补水

### 3. 赛后一周

| 天数 | 建议 |
|------|------|
| 1-3 天 | 休息或散步 |
| 4-7 天 | 轻松跑 20-30 分钟 |

### 4. 赛后调整

- 庆祝完成！
- 记录成绩和感受
- 分析什么做得好/不好
- 制定下一个目标

## 六、常见错误

| 错误 | 后果 | 正确做法 |
|------|------|----------|
| 起跑太快 | 后程崩溃 | 前程保守 |
| 不做减量 | 疲劳未消除 | 赛前 2-3 周减量 |
| 新装备比赛 | 磨损、不适 | 赛前磨合 50K+ |
| 跳过补给站 | 能量不足 | 按计划补给 |
| 不熟悉赛道 | 意外消耗 | 提前研究 |
