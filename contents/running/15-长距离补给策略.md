# 长距离补给策略

> 15K 以上的补给决定成败

## 一、为什么长距离需要补给？

### 1. 身体在长距离中的变化

| 时间 | 身体变化 |
|------|----------|
| 60 分钟 | 糖原开始耗尽 |
| 90 分钟 | 糖原明显不足 |
| 120 分钟 | 可能"撞墙" |
| 150+ 分钟 | 需要持续补给 |

### 2. 需要补给的内容

| 补给品 | 作用 | 何时需要 |
|--------|------|----------|
| 水 | 体温调节、代谢 | 始终 |
| 电解质 | 防止抽筋、保持体液平衡 | >60 分钟，高温 |
| 碳水化合物 | 维持血糖、能量 | >60 分钟 |
| 盐丸 | 补充钠、防止低钠血症 | 大量出汗时 |

## 二、补给计划

### 1. 根据距离制定

**10-15K**
```
补水：每 3-5K 喝水
碳水：通常不需要
电解质：高温时运动饮料
```

**15-21K（半马）**
```
补水：每 3-5K
碳水：45 分钟后开始，每 45 分钟 30g
电解质：运动饮料或盐丸
```

**30K+（全马训练）**
```
补水：每 3-5K
碳水：30 分钟后开始，每 45 分钟 30-60g
电解质：每小时盐丸 1-2 颗
```

**42K（全马比赛）**
```
补水：每站必喝
碳水：30 分钟开始，共 4-6 个能量胶
电解质：每小时盐丸
```

### 2. 根据时间制定

**< 60 分钟**
- 只需要水

**60-120 分钟**
- 每 45 分钟 30g 碳水
- 运动饮料或能量胶
- 每 20 分钟补水

**> 120 分钟**
- 每 45 分钟 30-60g 碳水
- 能量胶 + 能量棒/香蕉
- 盐丸每小时
- 运动饮料 + 水交替

## 三、补水策略

### 1. 跑前补水
```
跑前 2-3 小时：500ml
跑前 15-30 分钟：250ml
尿液颜色：淡黄色最佳
```

### 2. 跑中补水

**原则**：
- 每 20 分钟 150-200ml
- 小口多次，不要大口灌
- 口渴时立即喝

**补水站策略**：
| 距离 | 补给 |
|------|------|
| 5K | 水 |
| 10K | 水 + 运动饮料 |
| 15K | 水 + 能量胶 |
| 20K | 运动 + 盐丸 |
| ... | 循环 |

### 3. 跑后补水
```
体重流失 × 1.5 = 需补充水量
例：流失 1kg → 补充 1.5L
```

### 4. 过度补水的风险
- 低钠血症（血钠过低）
- 症状：头晕、恶心、头痛
- 预防：不要只喝白水，配合电解质

## 四、碳水补给

### 1. 能量胶

**优点**：
- 便携
- 快速吸收
- 精确剂量

**使用方法**：
- 每 45-60 分钟一个
- 配水服用
- 不要一次吃多个

**常见品牌**：
- GU
- PowerBar
- SIS
- Maurten

### 2. 能量棒

**优点**：
- 饱腹感
- 持续能量

**适合**：
- 超长距离（>30K）
- 训练中途

**缺点**：
- 可能口渴
- 消化较慢

### 3. 固体食物

**适合长距离/超马**：
- 香蕉
- 葡萄干
- 白面包
- 土豆
- 年糕

### 4. 碳水含量参考

| 食物 | 碳水含量 |
|------|----------|
| 能量胶 | 20-25g |
| 香蕉 | 25-30g |
| 能量棒 | 30-40g |
| 运动饮料 500ml | 30-35g |
| 葡萄干 30g | 22g |

## 五、电解质补给

### 1. 为什么需要电解质？

**钠的作用**：
- 保持体液平衡
- 防止抽筋
- 神经肌肉功能

**缺钠症状**：
- 抽筋
- 疲劳
- 头晕
- 恶心

### 2. 电解质来源

**运动饮料**：
- 方便
- 同时补碳水
- 选择等渗或高渗

**盐丸**：
- 便携
- 精确
- 大量出汗时必需

**天然来源**：
- 咸味零食
- 电解质粉

### 3. 盐丸使用

**何时需要**：
- 大量出汗（高温、高湿）
- 跑步 > 2 小时
- 容易抽筋的人

**用量**：
- 每小时 1-2 颗
- 配水服用
- 不要过量

## 六、补给训练

### 1. 训练中练习

**为什么重要**：
> 比赛时不要尝试新东西！

**训练内容**：
- 练习携带补给
- 练习跑步中进食
- 测试不同品牌
- 找到适合自己的

### 2. 补给计划制定

**比赛补给清单**：
```
赛前：
□ 跑前 2-3 小时早餐
□ 跑前 15 分钟水

赛中：
□ 能量胶 4-6 个
□ 盐丸 4-6 颗
□ 熟悉补给站位置

计划：
- 5K：水
- 10K：水 + 能量胶 1
- 15K：运动饮料
- 20K：水 + 能量胶 2 + 盐丸
- 25K：运动饮料
- 30K：水 + 能量胶 3 + 盐丸
- 35K：运动饮料 + 能量胶 4
- 40K：水
```

## 七、肠胃问题

### 1. 常见肠胃问题

| 问题 | 原因 | 解决 |
|------|------|------|
| 恶心 | 补给过多/过快 | 减慢补给速度 |
| 腹泻 | 高渗饮料/纤维 | 降低浓度，避免纤维 |
| 腹痛 | 空腹或太饱 | 调整进食时间 |

### 2. 预防肠胃问题

- 训练时练习补给
- 不要尝试新食物
- 补给小口多次
- 能量胶配水
- 比赛日早餐熟悉

### 3. 肠胃友好策略

**适合敏感肠胃**：
- 低纤维能量胶
- 多种糖源（葡萄糖+果糖）
- 小量多次
- 训练肠道适应
