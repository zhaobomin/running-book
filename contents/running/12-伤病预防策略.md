## 二、训练负荷管理

### 1. 10% 原则
每周跑量增加不超过 10%

**示例**：
| 周次 | 周跑量 | 增加量 |
|------|--------|--------|
| 第 1 周 | 20K | - |
| 第 2 周 | 22K | +10% |
| 第 3 周 | 24K | +9% |
| 第 4 周 | 20K | 减量周 |
| 第 5 周 | 26K | +8% |

### 2. 减量周
每 3-4 周安排一个减量周：
- 跑量减少 20-30%
- 让身体恢复
- 预防过度训练

### 3. 训练强度分配
```
80/20 原则：
- 80% 轻松跑
- 20% 高强度训练

很多跑者的问题是：
- 轻松跑不够轻松
- 高强度不够高
- 导致中间状态，效率低且受伤风险高
```

### 4. 监控训练负荷

**跑量记录**：
- 每周总距离
- 每周总时间
- 长距离距离

**疲劳信号**：
- 静息心率升高 5+ 次
- 睡眠质量下降
- 情绪烦躁
- 表现下降
- 频繁小伤病

## 三、恢复与休息

### 1. 休息日安排
| 跑步频率 | 适合人群 | 休息日 |
|----------|----------|--------|
| 3 天/周 | 新手 | 4 天 |
| 4-5 天/周 | 进阶 | 2-3 天 |
| 6 天/周 | 高水平 | 1 天 |
| 7 天/周 | 不推荐 | 0 天 |

### 2. 睡眠
**推荐**：每晚 7-9 小时

**睡眠的作用**：
- 肌肉修复
- 激素平衡
- 免疫力提升
- 心理恢复

**提高睡眠质量**：
- 固定作息
- 睡前 1 小时不用手机
- 保持卧室凉爽黑暗
- 避免睡前剧烈运动

### 3. 主动恢复

**比完全休息更好**：
- 轻松散步
- 游泳
- 骑车
- 瑜伽
- 泡沫轴放松

### 4. 营养恢复

**跑后 30-60 分钟内**：
- 碳水：蛋白质 = 3:1 或 4:1
- 如：香蕉 + 酸奶、巧克力奶

**日常营养**：
- 充足蛋白质（每公斤 1.4-1.7g）
- 均衡饮食
- 充足水分

## 四、力量与柔韧性

### 1. 力量训练

**每周 2-3 次，重点**：
- 臀中肌（稳定骨盆）
- 臀大肌（推蹬力量）
- 核心（躯干稳定）
- 小腿和脚踝（稳定落地）

**必做动作**：
1. 单腿站立（脚踝稳定）
2. 臀桥/单腿臀桥（臀肌）
3. 蚌式开合（臀中肌）
4. 平板支撑（核心）
5. 单腿硬拉（腘绳肌+臀肌）

### 2. 柔韧性训练

**每天可以做的拉伸**：
- 小腿拉伸
- 髋屈肌拉伸
- 腘绳肌拉伸
- 臀肌拉伸

**泡沫轴放松**：
- 大腿前侧
- 大腿后侧
- 小腿
- 髂胫束

### 3. 活动度

**髋关节活动度**：
- 90/90 拉伸
- 髋绕环
- 弓步转体

**脚踝活动度**：
- 脚踝绕环
- 小腿拉伸
- 足底筋膜放松

## 五、技术与装备

### 1. 跑姿要点

**预防伤病的关键**：
- 落地点在重心下方
- 步频 170-180
- 身体微前倾（从脚踝）
- 不跨步过大

### 2. 跑鞋选择

**更换周期**：500-800 公里

**选择原则**：
- 根据脚型选择
- 试穿时比日常鞋大半码
- 下午试鞋（脚会肿胀）
- 考虑跑步风格

**跑鞋类型**：
| 类型 | 适合人群 |
|------|----------|
| 缓震型 | 体重较大、足弓正常 |
| 稳定型 | 轻微过度内旋 |
| 控制型 | 严重过度内旋 |
| 赤足/极简 | 有经验的跑者 |

### 3. 训练场地

**场地硬度排序**（从软到硬）：
1. 草地
2. 森林步道
3. 橡胶跑道
4. 沥青
5. 水泥

**建议**：
- 大部分在较软地面
- 偶尔在硬地面（适应比赛）
- 下坡注意控制

## 六、自我监测

### 1. 晨起心率
- 每天早晨测量静息心率
- 如果比平时高 5+ 次，可能过度训练
- 考虑减量或休息

### 2. 主观疲劳度
```
1-10 分制：
1-3：非常轻松
4-6：中等
7-8：困难
9-10：非常困难

大部分训练应该在 4-6 分
```

### 3. 身体信号记录

**记录内容**：
- 睡眠质量
- 肌肉酸痛程度
- 精神状态
- 小伤痛

**每周回顾**：
- 是否持续疲劳
- 是否有新疼痛
- 是否需要调整

## 八、预防计划示例

### 一周预防性安排
```
周一：跑步 + 跑后拉伸
周二：力量训练 + 泡沫轴
周三：跑步 + 跑后拉伸
周四：休息或交叉训练
周五：力量训练 + 拉伸
周六：长距离 + 跑后拉伸
周日：完全休息或主动恢复
```
