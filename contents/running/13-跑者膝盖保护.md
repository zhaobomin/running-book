# 跑者膝盖保护

> 膝盖是跑者最脆弱的关节

## 一、为什么膝盖容易受伤？

### 1. 膝关节结构
```
股骨（大腿骨）
    ╲
     ╲  膝关节
      ╲
       ─── 髌骨（膝盖骨）
      ╱
     ╱
    ╱
胫骨（小腿骨）
```

### 2. 跑步时的冲击
- 每跑一步，膝盖承受体重 2-3 倍的冲击
- 跑 5K，膝盖承受约 5000 次冲击
- 肌肉力量不足时，关节承担更多压力

### 3. 膝盖疼痛的常见原因
| 原因 | 说明 |
|------|------|
| 臀中肌无力 | 导致膝盖内扣 |
| 股四头肌无力 | 髌骨轨迹异常 |
| 腘绳肌紧张 | 增加膝关节压力 |
| 跑量增加过快 | 超过适应能力 |
| 跑姿问题 | 跨步过大 |
| 鞋子问题 | 支撑不足 |

## 二、膝盖问题类型

### 1. 跑者膝（髌骨疼痛综合征）
**位置**：膝盖前方或髌骨周围

**症状**：
- 疼痛在膝盖前方
- 上下楼梯、下蹲时加重
- 久坐后站起来痛

### 2. 髂胫束综合征
**位置**：膝盖外侧

**症状**：
- 膝盖外侧疼痛
- 跑步时加重
- 下坡更明显

### 3. 髌腱炎
**位置**：髌骨下方

**症状**：
- 膝盖骨正下方疼痛
- 跳跃、跑步时加重

### 4. 半月板损伤
**位置**：膝盖内部

**症状**：
- 膝关节卡住感
- 深蹲时疼痛
- 可能有弹响

## 三、膝盖保护策略

### 1. 加强臀部肌肉

**为什么重要？**
> 臀中肌无力是跑者膝的首要原因

**训练动作**：

**蚌式开合**
```
目标：每侧 3 组 × 15-20 次

- 侧卧，膝盖弯曲 45°
- 保持脚跟接触
- 上方膝盖向上打开
- 感受臀部外侧发力

进阶：加弹力带
```

**侧卧抬腿**
```
目标：每侧 3 组 × 15 次

- 侧卧，下方腿弯曲
- 上方腿伸直，向上抬起
- 感受臀部外侧发力
```

**单腿站立**
```
目标：每侧 30 秒 × 3 组

- 单腿站立
- 保持髋部水平
- 膝盖对准脚尖
- 不要内扣

进阶：闭眼、不稳定平面
```

### 2. 加强股四头肌

**为什么重要？**
> 股四头肌是膝盖的主要稳定肌

**训练动作**：

**靠墙静蹲**
```
目标：30-60 秒 × 3 组

- 背靠墙
- 膝盖弯曲 90°
- 保持呼吸
- 感受大腿前侧发力

进阶：单腿靠墙静蹲
```

**终末伸膝**
```
目标：每侧 3 组 × 15 次

- 坐姿，膝盖弯曲
- 缓慢伸直膝盖
- 最后 30° 专注控制
- 缓慢放下
```

**台阶上下**
```
目标：每侧 3 组 × 12 次

- 站在台阶前
- 一脚踩上台阶
- 另一脚跟上
- 原路返回
- 控制 downward phase
```

### 3. 拉伸紧张肌肉

**股四头肌拉伸**
```
- 站立，手扶墙
- 一脚向后弯曲
- 手抓住脚踝
- 感受大腿前侧拉伸
- 保持 30 秒
```

**腘绳肌拉伸**
```
- 坐姿，一腿伸直
- 身体向前倾
- 手触脚尖
- 感受大腿后侧拉伸
- 保持 30 秒
```

**髂胫束放松**
```
- 侧卧，泡沫轴在大腿外侧
- 上下滚动
- 遇到痛点停留
- 每侧 2 分钟
```

### 4. 改善跑姿

**关键点**：
- 落地点在重心下方
- 步频提高（170-180）
- 避免跨步过大
- 膝盖微曲落地

### 5. 调整训练

**跑步调整**：
- 减少下坡跑
- 避免过多速度训练
- 考虑在软地面跑步
- 跑量增加放缓

**交叉训练**：
- 游泳
- 骑车
- 椭圆机
- 水中跑步

## 四、膝盖保护训练计划

### 每日（5 分钟）
```
1. 股四头肌拉伸 每侧 30 秒
2. 腘绳肌拉伸 每侧 30 秒
3. 髂胫束泡沫轴 每侧 1 分钟
```

### 跑后（10 分钟）
```
1. 单腿站立 每侧 30 秒
2. 蚌式开合 每侧 15 次
3. 靠墙静蹲 45 秒
4. 拉伸 5 分钟
```

### 力量训练日（每周 2-3 次）
```
1. 蚌式开合 每侧 3×20
2. 侧卧抬腿 每侧 3×15
3. 单腿臀桥 每侧 3×12
4. 靠墙静蹲 3×45秒
5. 台阶上下 每侧 3×12
6. 终末伸膝 每侧 3×15
```

## 五、膝盖疼痛时的处理

### 可以继续跑的情况
- 疼痛 ≤ 3/10
- 热身后疼痛消失
- 不影响跑姿
- 第二天没有加重

### 应该停止的情况
- 疼痛 > 3/10
- 影响跑姿
- 膝盖肿胀
- 疼痛持续加重

### 恢复策略
1. **休息**：停止跑步 2-7 天
2. **冰敷**：每天 2-3 次，每次 15 分钟
3. **力量训练**：继续做康复训练
4. **交叉训练**：游泳、骑车保持体能
5. **逐渐恢复**：从跑走开始

## 六、什么时候看医生

- 疼痛持续超过 2 周
- 膝关节肿胀
- 关节卡住或弹响
- 无法完全弯曲或伸直
- 疼痛影响日常生活
