# 速度训练方法

> 想要更快，必须练速度

## 一、速度训练基础

### 1. 为什么要练速度？

| 好处 | 说明 |
|------|------|
| 提高 VO2max | 最大摄氧量提升 |
| 提升跑步经济性 | 每步更省力 |
| 增强神经肌肉协调 | 跑得更流畅 |
| 提高配速 | 所有配速都会提升 |

### 2. 速度训练类型

| 类型 | 强度 | 目的 |
|------|------|------|
| 冲刺跑 | 95-100% | 神经肌肉 |
| 间歇跑 | 85-95% | VO2max |
| 节奏跑 | 80-85% | 乳酸阈值 |
| 法特莱克 | 变化 | 综合 |

### 3. 配速指南

| 训练类型 | 配速参考 | 心率区间 |
|----------|----------|----------|
| 轻松跑 | 比马拉松配速慢 45-60 秒 | 65-75% |
| 节奏跑 | 比马拉松配速快 10-15 秒 | 80-85% |
| 间歇跑 | 5K 比赛配速或更快 | 85-95% |
| 冲刺跑 | 几乎全力 | 95-100% |

## 二、间歇跑

### 1. 什么是间歇跑？

**定义**：高强度跑与恢复期交替进行

**核心原理**：
- 高强度刺激心肺
- 间歇期不完全恢复
- 累积效果提升 VO2max

### 2. 经典间歇训练

**400 米间歇**
```
热身：10 分钟轻松跑 + 动态拉伸
主训练：400m × 8-12
配速：5K 比赛配速或稍快
间歇：慢跑或走 200m（约 1-2 分钟）
冷身：10 分钟轻松跑
```

**800 米间歇**
```
热身：15 分钟
主训练：800m × 4-6
配速：5K 比赛配速
间歇：慢跑 400m（约 2-3 分钟）
冷身：10 分钟
```

**1 公里间歇**
```
热身：15 分钟
主训练：1K × 3-5
配速：5K 比赛配速
间歇：慢跑 400-800m（约 3 分钟）
冷身：10-15 分钟
```

### 3. 间歇训练进阶

**增加方式**：
- 增加重复次数
- 缩短间歇时间
- 提高配速
- 增加组数

**示例进阶**：
```
第 1 周：400m × 6
第 2 周：400m × 8
第 3 周：400m × 10
第 4 周：减量（400m × 6）
第 5 周：400m × 12 或 800m × 6
```

## 三、节奏跑

### 1. 什么是节奏跑？

**定义**：在"舒适地困难"强度持续跑步

**特点**：
- 能说出短语，不能完整对话
- 配速稳定
- 持续 20-40 分钟

### 2. 节奏跑训练

**标准节奏跑**
```
热身：10-15 分钟轻松跑
主训练：20-30 分钟节奏跑
配速：比 10K 比赛配速慢 10-15 秒/公里
冷身：10 分钟轻松跑
```

**节奏间歇**
```
热身：15 分钟
主训练：15 分钟 × 2 组
间歇：3 分钟轻松跑
冷身：10 分钟
```

**渐进节奏跑**
```
热身：10 分钟
主训练：
- 10 分钟稍慢节奏
- 10 分钟目标节奏
- 10 分钟稍快节奏
冷身：10 分钟
```

### 3. 节奏跑配速确定

**方法一：比赛成绩推算**
| 5K 时间 | 节奏跑配速 |
|---------|------------|
| 25:00 | 5:15-5:25/公里 |
| 22:00 | 4:40-4:50/公里 |
| 20:00 | 4:15-4:25/公里 |

**方法二：体感判断**
- 能说 2-3 个词
- 呼吸有节奏但较重
- 感觉"舒适地困难"

## 四、法特莱克

### 1. 什么是法特莱克？

**定义**：瑞典语"速度游戏"，非结构化的速度训练

**特点**：
- 灵活自由
- 根据体感调整
- 有趣多变

### 2. 法特莱克训练

**基础法特莱克**
```
热身：10-15 分钟轻松跑
主训练：
- 快跑 2 分钟
- 慢跑 2 分钟
- 快跑 1 分钟
- 慢跑 1 分钟
- 重复 3-4 轮
冷身：10 分钟
```

**地标法特莱克**
```
热身：10 分钟
主训练：
- 跑到那棵树（快）
- 跑到那个转角（慢）
- 跑到那个路牌（快）
- 自由发挥 20-30 分钟
冷身：10 分钟
```

## 五、冲刺跑

### 1. 什么是冲刺跑？

**定义**：接近全力的短距离跑

**目的**：
- 神经肌肉训练
- 提高跑步经济性
- 建立速度储备

### 2. 冲刺训练

**斜坡冲刺**
```
热身：15 分钟轻松跑 + 动态拉伸
主训练：上坡冲刺 10-15 秒 × 8-10 次
休息：走下坡（约 1-2 分钟）
冷身：10 分钟轻松跑
```

**平地冲刺**
```
热身：15 分钟 + 加速跑
主训练：100m 冲刺 × 6-8 次
休息：走回起点（约 2 分钟）
冷身：10 分钟
```

**跑后冲刺**
```
轻松跑结束后：
- 4-6 × 20-30 秒快跑
- 休息 1 分钟
- 不是全力，是"快但控制"
```

## 六、速度训练安排

### 1. 每周安排

**基础安排**：
| 天数 | 训练 |
|------|------|
| 周一 | 休息 |
| 周二 | 间歇跑 |
| 周三 | 轻松跑 |
| 周四 | 节奏跑 |
| 周五 | 休息 |
| 周六 | 长距离 |
| 周日 | 休息 |

**进阶安排**：
| 天数 | 训练 |
|------|------|
| 周一 | 轻松跑 |
| 周二 | 间歇跑 |
| 周三 | 轻松跑 |
| 周四 | 节奏跑 |
| 周五 | 休息 |
| 周六 | 长距离 + 冲刺 |
| 周日 | 休息 |

### 2. 训练周期

**基础期**：
- 重点：轻松跑 + 少量节奏跑
- 速度训练：每周 1 次

**强化期**：
- 重点：间歇 + 节奏
- 速度训练：每周 2 次

**赛前减量**：
- 保持强度
- 减少量
- 速度训练：每周 1 次

## 七、注意事项

### 1. 安全原则

- 充分热身
- 循序渐进
- 不连续两天高强度
- 身体不适就停止

### 2. 常见错误

| 错误 | 正确做法 |
|------|----------|
| 间歇跑太快 | 按目标配速，不是全力 |
| 间歇时间太短 | 保证充分恢复 |
| 节奏跑太累 | "舒适地困难"，不是拼命 |
| 热身不足 | 至少 10-15 分钟 |
