# 《跑步训练与实践》写作风格与统一口径（初始化治理版）

版本：2026-03-02

## 1. 受众画像
- A 类：零基础/回归跑者（目标：安全起步、建立习惯）。
- B 类：5K/10K 进阶跑者（目标：稳定完赛、逐步提速）。
- C 类：半马/全马备赛跑者（目标：系统训练、稳定完赛或 PB）。
- D 类：高风险读者（慢病史、近期伤病、睡眠长期不足、体重波动明显）。

统一要求：每章开头明确“适用人群”和“不适用人群”。

## 2. 语气与表达
- 语气：教练式、克制、清晰、可执行。
- 句式：优先短句和行动句，避免空泛口号。
- 强制：结论后跟边界条件（适用场景/禁忌场景/替代方案）。
- 禁止：无条件绝对化表达（如“必须”“唯一”“所有人都应”）。

## 3. 术语表（统一口径）
- 轻松跑：可完整对话，通常用于有氧基础或恢复。
- 节奏跑：舒适地困难，能说短语不能完整对话。
- 间歇跑：高强度重复 + 恢复段交替。
- 长距离跑：以耐力适应为目标，不以速度测试为目标。
- 减量期：赛前降低训练量、保留部分强度的阶段。
- 最大心率（HRmax）、静息心率（RHR）、心率变异性（HRV）。
- 红旗信号：应立即停训并考虑就医的症状/体征。

## 4. 统一单位与格式
- 配速：`分:秒/公里`。
- 距离：`公里（km）`。
- 心率：`次/分钟（bpm）`。
- 温度：`°C`。
- 营养：`g/kg 体重`、液体补给用 `ml`。
- 时间敏感信息：加 `截至 YYYY-MM-DD，需复核`。

## 5. 章节结构模板（强制）
1. 本章目标（1-3 条）
2. 适用人群与前置条件
3. 核心方法与步骤
4. 可执行计划（周计划/清单/流程）
5. 风险与边界（含“非医疗建议”）
6. 何时停止训练并就医（红旗信号）
7. FAQ（聚焦执行问题）
8. 参考来源（可追溯）

## 6. 示例标准
- 合格示例：
  - “若连续 2 次训练在前 1/3 出现动作失控，下次配速下调 5-10 秒/公里。”
  - “赛前 3 天碳水建议 7-10 g/kg；糖代谢异常人群需先咨询医生。”
- 不合格示例：
  - “多练就会进步。”
  - “赛前一定要猛吃碳水。”

## 7. 事实与引用表达
- 出现“研究表明/可降低X%/建议摄入Y g/kg”必须附来源。
- 无来源时标注“需确认”，不得补写猜测数据。
- 不得将经验值写成普适事实。

## 8. 重复与一致性
- 同章重复标题、重复 FAQ、重复表格属于严重质量问题。
- 跨章术语冲突必须解释适用条件，不得并列给出矛盾结论。

## 9. 包容性与边界
- 不羞辱配速、体重、年龄、性别。
- 生理期、中老年、伤后回归人群提供替代方案，不一刀切。
